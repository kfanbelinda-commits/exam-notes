# 金融计量学 · 所有检验总整理

> 按知识逻辑线组织，每个检验写在它该出现的步骤里。
> 每个检验都回答：**检验什么 → H₀ → 怎么判断 → 结论 → 后续怎么办**

## 第一条线：OLS 基础——古典假定有没有被违反？

> 背景：OLS 估计出 β̂ 之后，β̂ 的好性质（无偏、最小方差、一致）依赖古典假定全部成立。
> 所以估计完 β̂ 要做"体检"，逐项检查。发现问题 → 修正 → 修正后才能放心做推断。

### 1.1 显著性检验（模型估计完，先看结果能不能用）

#### t 检验

- 检验什么：**单个**解释变量对Y有没有显著影响（系数是否显著异于零）
- H₀：βᵢ = 0（这个变量对Y没有解释力）
- H₁：βᵢ ≠ 0
- 统计量：t = β̂ᵢ / se(β̂ᵢ) ~ t(n−k−1)
- 判断：**双侧**检验 → 临界值查 α/2（如 α=0.05 查 t₀.₀₂₅）→ |t统计量| vs 临界值
  - |t| > 临界值 → 拒绝H₀ → 显著
  - |t| < 临界值 → 不拒绝H₀ → 不显著
- ⚠ t检验是唯一的双侧检验（F和卡方都是单侧）
- ⚠ 实操简化：对t统计量取绝对值 → 转为单侧处理 → 统一套"数大率小拒H₀"

#### F 检验

- 检验什么：**所有**解释变量联合起来对Y有没有显著影响（整个方程是否显著）
- H₀：β₁ = β₂ = … = βₖ = 0（所有变量联合起来没有解释力）
- H₁：β₁, β₂, …, βₖ 不全为零
- 统计量：F = [R²/k] / [(1−R²)/(n−k−1)] ~ F(k, n−k−1)
- 判断：**单侧**检验 → 临界值直接查 α → F统计量 vs 临界值
  - F > 临界值 → 拒绝H₀ → 方程整体显著
- ⚠ 一元回归时 F = t²，两者等价；多元回归时各管各的
- ⚠ 多重共线性时 t 检验失效但 F 不受影响（各变量单独不显著，联合起来显著）

#### 受约束 F 检验（Chow 检验是其特例）

- 检验什么：一组约束条件是否成立（如"两段数据的系数相同"）
- H₀：约束成立（如两段参数相同）
- 统计量：F = [(RSS_R − RSS_UR)/q] / [RSS_UR/(n−k−1)]
  - q = 约束个数；RSS_R = 受约束模型的残差平方和（≥ RSS_UR）
- Chow 检验：约束 = "两段时间的系数完全相同"，用于检验结构突变

### 1.2 同方差假定 → 异方差检验（5种）

> 检验什么：残差的方差是不是前后一致。如果方差随X变化（异方差）→ OLS标准误算错 → t/F不可靠。

#### ① 散点图法

- 做法：画残差²对X的散点图
- 判断：有喇叭口（方差随X扩大/缩小）→ 可能有异方差
- 特点：最直观，但不严格

#### ② GQ 检验（Goldfeld-Quandt）

- 做法：把数据按X大小排序，分成两段，分别做OLS得 RSS₁ 和 RSS₂，构造 F = RSS₂/RSS₁
- H₀：同方差
- 判断：F > 临界值 → 两段方差差异显著 → 有异方差
- 适用：递增型或递减型异方差
- ⚠ GQ 是异方差检验，不是自相关检验（选择题考过）

#### ③ White 检验（怀特检验）★最通用

- 做法：残差²对X及其交叉项、平方项做辅助回归，构造 nR² ~ χ²
- H₀：不存在异方差（同方差）
- 判断：看 **Obs*R-squared 行**的 p 值
  - p < 5% → 拒绝H₀ → 有异方差
  - p > 5% → 不拒绝H₀ → 同方差
- 优点：不需要假设异方差的具体形式
- ⚠ 读表看第二行 Obs*R-squared，不看F那行（避免临界值附近结论翻转）

#### ④ Glejser 检验（格里瑟检验）

- 做法：|残差| 对X做回归
- H₀：同方差
- 特点：可以推断异方差的函数形式（递增/递减/U型等）

#### ⑤ BP 检验（Breusch-Pagan）

- 做法：残差²对解释变量做回归 → nR² ~ χ²
- H₀：不存在异方差
- 和White类似，但不加交叉项和平方项，更简洁

#### 异方差的后果与修正

- 后果：OLS仍无偏一致，但不再最小方差（BLUE被打破）→ 标准误算错 → t/F不可靠
- 修正：WLS（加权最小二乘）/ 怀特稳健标准误（不改β̂，只修正标准误）

### 1.3 无自相关假定 → 自相关检验（2种）

> 检验什么：残差前后期之间有没有关联。如果有自相关 → 同样导致标准误算错。

#### ① DW 检验（Durbin-Watson）

- 检验什么：残差有没有**一阶**自相关
- 公式：DW ≈ 2(1−ρ̂)，ρ̂ = 相邻残差的相关系数
- H₀：无自相关
- 判断：不需要看p值，直接看DW值落在哪个区间
  - DW = 0 → ρ̂=1 → 完全正自相关
  - DW = 2 → ρ̂=0 → 无自相关（好消息）
  - DW = 4 → ρ̂=−1 → 完全负自相关
  - 区间：0~dL 正相关 / dL~dU 不确定 / dU~4-dU 无相关 / 4-dU~4-dL 不确定 / 4-dL~4 负相关
- 限制：只能检验一阶自相关；要求模型含截距项、不含被解释变量滞后项
- ⚠ DW 是自相关检验，不是异方差检验

#### ② BG-LM 检验（Breusch-Godfrey）

- 检验什么：残差有没有**高阶**自相关（比DW更通用）
- 做法：残差对自身滞后项做自回归 → nR² ~ χ²
- H₀：不存在自相关
- 判断：看 **Obs*R-squared 行**的 p 值
  - p < 5% → 有自相关
  - p > 5% → 无自相关

#### 自相关的后果与修正

- 后果：和异方差一样——OLS仍无偏，但标准误有偏，t/F不可靠
- 修正：广义差分 / GLS / Cochrane-Orcutt迭代法
- ⚠ 金融计量学恰恰要利用自相关 → 建AR/MA/ARMA模型（第二条线）

### 1.4 无多重共线性假定

> 检验什么：解释变量之间有没有高度线性相关。

#### 判断方法（4种，不是假设检验的形式）

- ① 简单相关系数 |Rᵢⱼ| ≈ 1 → 两个变量高度相关
- ② VIF > 5 或 > 10 → 严重共线性
- ③ F大、R²大、但各个t都小 → 联合显著但各自不显著 → 共线性的典型表现
- ④ 条件指数 CI > 20

#### 后果与修正

- 完全共线性：X'X不可逆 → 无法求出β̂
- 不完全共线性：无偏一致，但方差膨胀 → t失效 → 预测精度下降
- ⚠ F检验和R²不受影响（和异方差/自相关不同）
- 修正：剔除不重要变量 / 增加样本 / 逐步回归 / 岭回归

### 1.5 外生性假定 → 内生性检验

> 检验什么：解释变量X和扰动项U有没有相关。内生性是最严重的违反——OLS有偏且不一致，样本量再大也救不回来。

#### Hausman 检验

- 检验什么：有没有内生性
- H₀：不存在内生性（Cov(X, U) = 0）
- 做法：分别做OLS和IV（工具变量法）估计，比较两个β̂
  - 差距不大 → 没有内生性，OLS就够了
  - 差距显著 → 有内生性，必须用IV
- 判断：检验统计量显著 → 拒绝H₀ → 有内生性

#### 后果与修正

- 后果：OLS**有偏且不一致**（这是唯一破坏无偏性的违反，其他违反只影响效率）
- 修正：工具变量法 / 2SLS
  - 找工具变量Z：Cov(Z,X)≠0（相关性）+ Cov(Z,u)=0（外生性）
  - 第一步：Z回归X → 得到干净的X̂
  - 第二步：用X̂替代X对Y做OLS → 得到一致的β̂

### 1.6 假设违反后果对比（一张表记住）

| 违反类型 | 无偏性 | 一致性 | 最小方差 | t/F检验 | 修正方法 |
|---------|--------|--------|---------|---------|---------|
| 多重共线性 | ✓保持 | ✓保持 | ✗打破 | t失效，F不受影响 | 剔除变量/岭回归 |
| 异方差 | ✓保持 | ✓保持 | ✗打破 | 都不可靠 | WLS/稳健标准误 |
| 自相关 | ✓保持 | ✓保持 | ✗打破 | 都不可靠 | 广义差分/GLS |
| 内生性 | **✗有偏** | **✗不一致** | ✗打破 | 都不可靠 | IV/2SLS |

## 第二条线：时间序列建模——从原始序列到建模完成

> 背景：金融数据（股价、收益率）是时间序列，有自相关性和波动聚集现象。
> 时间序列建模 = 利用这些特性做预测，而不是把它们当作"违反假设"去修正。
> 整条线是一个固定流程，每一步用到的检验不同。

### 2.1 第一步：平稳性检验（序列能不能直接用？）

> 为什么要检验：平稳是建模的前提。不平稳序列直接回归 → 伪回归（R²虚高但无真实关系）。

#### 三种方法（从直观到严格）

**① 时间路径图法**
- 做法：直接画序列的时序图
- 判断：有明显趋势或方差变化 → 可能不平稳
- 特点：最直观，但不严格

**② 自相关系数法（ACF图）**
- 做法：画ACF图
- 判断：ACF缓慢衰减（拖很长的尾巴）→ 不平稳；ACF快速衰减 → 平稳
- 特点：比看图严格一点，但仍是辅助判断

**③ 单位根检验（DF / ADF）**★ 最严格，考试必用

- 检验什么：序列有没有单位根（单位根 = 非平稳）
- **DF 检验**：针对AR(1)，前提是残差无自相关
  - H₀：φ = 0（有单位根，非平稳）
  - H₁：φ < 0（无单位根，平稳）
  - ⚠ 单侧检验（只看φ<0这一个方向）
- **ADF 检验**：DF的高阶拓展，在方程中加入ΔY滞后项消除残差的高阶自相关
  - H₀、H₁同DF
  - 残差无自相关 → DF够用；残差有自相关 → 必须用ADF
- 判断：|ADF统计量| > 临界值 → 拒绝H₀ → **平稳**（好消息）
- ⚠ 拒绝H₀才是好消息——方向反直觉，和大多数检验相反
- ⚠ ADF使用特殊DF分布的临界值，不是标准t分布

#### 不平稳怎么办？

- 做差分：ΔYₜ = Yₜ − Yₜ₋₁
- 差几次变平稳就叫几阶单整 I(d)
- 实际中常用对数差分：ln(Pₜ) − ln(Pₜ₋₁) ≈ 收益率

### 2.2 第二步：模型识别（序列是什么类型的自相关？）

> 平稳之后，要确定自相关是什么形式的，才能选对模型。

#### ACF / PACF 识别

| ACF | PACF | 选什么模型 |
|-----|------|----------|
| 拖尾 | **截尾**（p阶后为零）| AR(p) |
| **截尾**（q阶后为零）| 拖尾 | MA(q) |
| 拖尾 | 拖尾 | ARMA(p,q) |

- 记忆口诀：AR→PACF截，MA→ACF截，ARMA→全拖
- 截尾 = 某阶后全部落入置信区间；拖尾 = 衰减但不为零

#### 定阶（p和q具体是几）

- 方法一：看图，几阶后进入置信区间 → 就是几阶
- 方法二：信息准则，比较候选模型的AIC/BIC，取最小
  - AIC = ln(σ̂²) + 2k/n；BIC = ln(σ̂²) + k·ln(n)/n
  - BIC惩罚更重，AIC与BIC不一致时优先BIC

#### 平稳性补充

- MA模型 → 一定平稳，不用再检验
- AR模型 → 需要看特征根
  - 写特征方程 → 求根 → 所有根的模 > 1（在单位圆外）→ 平稳
  - 有根 = 1 → 单位根 → 非平稳
  - ⚠ 根z = 1/φ，|φ|<1（衰减）等价于 |z|>1（圆外）

### 2.3 第三步：残差白噪声检验（模型拟合好不好？）

> 建完模型，提取残差。残差是白噪声 → 说明模型已经把确定性信息提取干净了 → 拟合好。
> 残差不是白噪声 → 还有信息没被提取 → 需要调整模型阶数。

#### 白噪声三条件

- ① 均值恒定（E(εₜ) = μ）
- ② 方差恒定（Var(εₜ) = σ²）
- ③ 完全无自相关（Cov(εₜ, εₛ) = 0, t≠s）

#### 检验方法

**① ACF图观察法**
- 做法：画残差的ACF图
- 判断：所有ACF都在置信区间内 → 白噪声

**② Box-Pierce 检验（Q_BP）**
- 统计量：Q_BP = n · Σρ̂ₖ² ~ χ²(m)
- H₀：残差是白噪声
- 判断：Q_BP < 临界值（数小率大）→ 不拒绝H₀ → 白噪声 → 模型OK

**③ Ljung-Box 检验（Q_LB）**★ BP的改进版，更准
- 统计量：Q_LB = n(n+2) · Σ[ρ̂ₖ²/(n−k)] ~ χ²(m)
- H₀：残差是白噪声
- 判断：p > 5% → 不拒绝H₀ → 白噪声 → 模型OK
- ⚠ 白噪声检验的H₀方向特殊：H₀是"好消息"（是白噪声），**不拒绝H₀**才是好结果

### 2.4 第四步：ARCH 效应检验（波动率需不需要单独建模？）

> 金融数据的残差可能均值为零、无自相关（看起来像白噪声），但方差不恒定——大波动之后还是大波动（波动聚集）。
> 这时候需要对方差也建模（ARCH/GARCH）。

#### ARCH-LM 检验

- 检验什么：残差的平方 ε̂²ₜ 有没有自相关（ε̂²ₜ 有自相关 = 波动率可预测 = 有ARCH效应）
- 做法：ε̂²ₜ 对自身滞后项做自回归 → nR² ~ χ²(q)
- H₀：不存在ARCH效应（所有αᵢ = 0）
- 判断：
  - 拒绝H₀ → 有ARCH效应 → 需要建GARCH模型
  - 不拒绝H₀ → 无ARCH效应 → 方差恒定 → 建模完成

#### 也可以用 Q_BP / Q_LB 检验ε̂²ₜ

- 对ε̂²ₜ（不是εₜ本身）做白噪声检验
- ε̂²ₜ是白噪声 → 波动无规律 → 无ARCH效应
- ε̂²ₜ不是白噪声 → 波动有规律 → 有ARCH效应 → 建GARCH

#### ARCH vs GARCH 识别

- 方差方程只有 ε²项 → ARCH(q)
- 方差方程同时有 ε²项 + h项（方差自身滞后） → GARCH(p,q)
- 读参数：看求和符号上界，ε²的上界 = q，h的上界 = p
- GARCH(1,1)参数约束：ω > 0，α ≥ 0，β ≥ 0，α + β < 1（弱平稳）

### 2.5 完整流程图（一张图串起来）

```
原始序列
  ↓
平稳性检验（ADF）
  ├─ 不平稳 → 差分 → 再检验 → 直到平稳
  └─ 平稳 ↓
ACF/PACF 识别 → 建 ARMA(p,q)（均值方程）
  ↓
提取残差 ε̂ₜ
  ↓
白噪声检验（LB）
  ├─ 不是白噪声 → 调整模型阶数 → 重新建模
  └─ 是白噪声 ↓
ARCH-LM 检验（对 ε̂²ₜ）
  ├─ 有ARCH效应 → 建 GARCH(p,q)（方差方程）
  └─ 无ARCH效应 → 建模完成
```

## 第三条线：多序列拓展——两个以上的序列之间有没有关系？

> 背景：第二条线处理的是单个序列。如果有两个以上的序列（比如收入和消费），想研究它们之间的关系，就走这条线。

### 3.1 协整检验（有没有长期均衡关系？）

> 两个序列各自不平稳（都是I(1)），能不能回归？不能直接回归（会伪回归），但如果它们有协整关系，就可以。

#### EG 两步法（一阶协整：Y和单个X）★重点

- 检验什么：两个I(1)序列之间有没有长期稳定关系
- 第一步：对Y和X做OLS → 得到残差 û = Y − Ŷ
- 第二步：对残差û做ADF检验
- H₀：无协整（残差不平稳）
- H₁：有协整（残差平稳）
- 判断：残差平稳 → 拒绝H₀ → 存在一阶协整
- 经济含义：残差平稳说明Y和X的偏差长期会回归均值，存在共同变动趋势
- ⚠ 此处ADF临界值与普通ADF不同（残差是估计出来的，标准更严格）
- ⚠ 两个I(1)不代表一定有协整，必须做EG检验才知道

#### Johansen 检验（多阶协整：多个变量）

- 用在：三个以上的变量之间的协整
- 了解名字即可，大部分学校不考具体操作

### 3.2 误差修正模型 ECM（短期偏离怎么调整回来？）

> 确认有协整之后的下一步：有长期关系了，短期偏离了怎么办？

- 模型：ΔYₜ = α·ûₜ₋₁ + 短期动态项 + vₜ
- ûₜ₋₁ = Yₜ₋₁ − β̂Xₜ₋₁（上一期偏离长期均衡的程度）
- α < 0：上期Y偏高 → 本期ΔY向下调整（纠偏回均衡）
- 步骤：第一步OLS建长期关系 → 第二步建ECM描述短期调整
- 一句话：EG告诉你"有没有长期关系"，ECM告诉你"偏离了怎么回来"

### 3.3 格兰杰因果检验（谁能预测谁？）

> 确认有协整之后还想知道：是X引起Y，还是Y引起X？

- 检验什么：在已用Y自身滞后项的基础上，加入X的滞后项能否**额外**提高对Y的预测
- 方程：Yₜ = c + Σβⱼ·Yₜ₋ⱼ + Σγⱼ·Xₜ₋ⱼ + uₜ
- H₀：γ₁ = γ₂ = … = γₚ = 0（X的滞后项系数全为零 → X对Y无增量预测力）
- 判断：F检验
  - 拒绝H₀ → X是Y的格兰杰原因
  - 不拒绝H₀ → X不是Y的格兰杰原因
- 四种情况（看交叉项α和λ）：

| 条件 | 结论 |
|------|------|
| α ≠ 0，λ = 0 | X是Y的格兰杰原因 |
| α = 0，λ ≠ 0 | Y是X的格兰杰原因 |
| α ≠ 0，λ ≠ 0 | 互为格兰杰原因 |
| α = 0，λ = 0 | 无格兰杰因果 |

- ⚠ 格兰杰因果是统计预测关系，不是真实因果
- ⚠ 结论依赖模型设定（滞后阶数、有没有遗漏第三变量）

### 3.4 多序列完整流程

```
两个序列各自 I(1)
  ↓
EG 检验：有没有协整？
  ├─ 无协整 → 伪回归，结论不可信，停止
  └─ 有协整 ↓
      ├─ 建 ECM：描述短期调整机制
      └─ 格兰杰因果检验：谁能预测谁？
```

## 附：H₀ 方向速查表

> 所有检验的H₀一张表查完，不用每次回去翻。

### 第一类：诊断性检验 → H₀ = "不存在XX"（没毛病）

| 检验 | H₀ | 拒绝H₀意味着 |
|------|-----|-----------|
| White/BP/GQ/Glejser | 不存在异方差 | 有异方差，需修正 |
| DW/BG-LM | 不存在自相关 | 有自相关，需修正 |
| Hausman | 不存在内生性 | 有内生性，用IV |
| ARCH-LM | 不存在ARCH效应 | 有ARCH效应，建GARCH |

记法：完美是小概率 → "没毛病"放H₀ → 拒绝才说明有问题

### 第二类：显著性检验 → H₀ = "等于零"（没有影响）

| 检验 | H₀ | 拒绝H₀意味着 |
|------|-----|-----------|
| t检验 | βᵢ = 0 | 该变量显著 |
| F检验 | β₁=…=βₖ=0 | 方程整体显著 |
| 格兰杰因果 | γ₁=…=γₚ=0 | X是Y的格兰杰原因 |

记法：系数为零 = 这个变量可有可无 → 拒绝 = 有用

### 第三类：特殊方向 → 单独记

| 检验 | H₀ | 好消息是什么 |
|------|-----|----------|
| ADF | 非平稳（有单位根）| **拒绝**H₀ = 平稳 |
| 白噪声(LB/BP) | 是白噪声 | **不拒绝**H₀ = 残差OK |
| EG协整 | 无协整 | **拒绝**H₀ = 有长期关系 |

## 附：白噪声的三大应用场景

> 同一个工具（白噪声检验），用在不同对象上，问的问题不同。

| 对什么做 | 问的问题 | 是白噪声意味着 | 不是白噪声意味着 |
|---------|---------|-------------|--------------|
| 对平稳序列做 | 序列能不能预测？ | 完全随机，不值得建模 | 有自相关，可以建模 |
| 对模型残差做 | 模型拟合好不好？ | 信息提取干净，拟合好 | 还有信息没提取，调整模型 |
| 对ε²做 | 波动率需不需要建模？ | 方差恒定，无ARCH效应 | 有ARCH效应，需建GARCH |
