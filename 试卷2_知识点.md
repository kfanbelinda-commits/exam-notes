## 一、填空题

### 填空Q1：平稳性检验的方法有____、____和____

- **答案**：时间路径图法、自相关系数法（ACF）、单位根检验法（DF/ADF）
- **知识点**：
  - 用在哪：拿到一个时间序列，第一步就是看它平不平稳——平稳才能建模预测，不平稳直接回归会伪回归
  - 三种方法从直观到严格：
    - ① 时间路径图法：直接画图看序列有没有明显趋势或方差变化
    - ② 自相关系数法：看ACF衰减速度，平稳序列ACF快速衰减，非平稳序列ACF缓慢衰减
    - ③ 单位根检验法：DF/ADF，构造统计量做假设检验，最严格

### 填空Q2：单位根检验的方法有____和____

- **答案**：DF检验、ADF检验
- **知识点**：
  - DF检验：针对AR(1)，前提是残差无自相关
  - ADF检验：DF的拓展，在方程中加入ΔY的滞后项来消除残差的高阶自相关
  - 关系：ADF是DF对高阶情况的推广，考试大部分用ADF

### 填空Q3：残差项不存在自相关时用____，存在自相关时用____

- **答案**：DF检验、ADF检验
- **知识点**：
  - 为什么要区分：DF假设残差是干净的（无自相关），如果残差本身有自相关，DF的统计量就不准了
  - ADF的做法：在DF方程基础上加入ΔYₜ₋₁, ΔYₜ₋₂ 等滞后项，把残差的自相关"吸收"掉
  - 判断标准：先看残差有没有自相关 → 没有用DF就行 → 有则必须用ADF

### 填空Q4：EG检验拒绝零假设说明____

- **答案**：被检验变量之间存在协整关系
- **知识点**：
  - EG检验是什么：Engle-Granger两步法，检验两个I(1)序列之间有没有长期稳定关系（协整）
  - H₀：无协整（残差不平稳）；H₁：有协整（残差平稳）
  - 拒绝H₀ = 残差平稳 = 两个序列之间有协整 = 好消息
  - ⚠ "拒绝零假设 → 有协整"，不要反过来

### 填空Q5：DF检验的零假设是说被检验时间序列____

- **答案**：非平稳（存在单位根）
- **知识点**：
  - H₀放的是"坏消息"（非平稳），拒绝H₀才是好消息（确认平稳）
  - 为什么这么设：假设检验的框架是"先假设H₀为真，看数据够不够反对它"。把非平稳放在H₀，一旦拒绝就能积极下结论"序列平稳"
  - ⚠ 方向反直觉——大多数检验H₀是"没毛病"，但ADF的H₀是"有问题"（非平稳）

### 填空Q6：协整性检验方法有____和____

- **答案**：EG检验（一阶协整）、JJ检验/Johansen检验（多阶协整）
- **知识点**：
  - 用在哪：多个序列各自不平稳，但想研究它们之间有没有长期关系
  - 一阶协整（Y和单个X）→ EG两步法：①OLS得残差 û = Y − Ŷ → ②对û做ADF检验
  - 多阶协整（Y和多个X/含滞后项）→ Johansen检验
  - ⚠ 区分：平稳性检验（DF/ADF）针对单个序列；协整检验（EG/JJ）针对多个序列之间的关系

### 填空Q7：用一个时间序列对另一个回归，无真实关系但R²很高，存在____问题

- **答案**：伪回归
- **知识点**：
  - 什么是伪回归：两个非平稳序列各自有趋势（比如都在涨），直接回归会因为共同趋势而显示出虚假的高R²
  - 根本原因：非平稳序列的共同趋势导致虚假相关，不代表两者有真实经济关系
  - 判断标志：R² > DW 时高度怀疑伪回归（残差高度自相关说明是趋势驱动的）
  - 怎么避免：先做协整检验，确认有真实的长期关系后才能回归

### 填空Q8：建立误差校正模型的步骤：第一步____，第二步____

- **答案**：第一步建立长期关系模型（OLS得协整方程）；第二步建立短期动态关系即误差校正方程
- **知识点**：
  - ECM用在哪：确认两个变量有协整关系（EG检验通过）之后，想研究短期内偏离长期均衡时怎么调整回去
  - 例子：收入和消费是协整的（长期消费倾向稳定），某个季度消费突然偏高 → ECM描述下个季度怎么往回调
  - 模型：ΔYₜ = α·ûₜ₋₁ + 短期动态项 + vₜ
    - ûₜ₋₁ = Yₜ₋₁ − β̂Xₜ₋₁（上一期偏离长期均衡的程度）
    - α < 0：上期Y偏高 → 本期ΔY向下调整（纠偏回均衡）
  - 一句话：EG检验告诉你"有没有长期关系"，ECM告诉你"偏离了怎么回来"
  - ⚠ 流程：先EG检验确认协整 → 再建ECM描述短期调整机制

## 二、单项选择题

### 单选Q1：某时间序列经一次差分变换成平稳时间序列，称为？

- **答案**：A. 1阶单整
- **知识点**：
  - 差几次变平稳就叫几阶单整：I(d)中d = 差分次数
  - 一次差分变平稳 → I(1)，即1阶单整
  - 两次差分变平稳 → I(2)，即2阶单整
  - 本身就平稳 → I(0)

### 单选Q2：如果两个变量都是一阶单整，则？

- **答案**：D. 还需对误差项进行检验
- **知识点**：
  - 两个I(1)变量不代表一定有协整关系——你还得检验
  - 怎么检验：EG两步法
    - ① OLS得残差 û = Y − Ŷ
    - ② 对û做ADF检验，看残差平不平稳
  - 残差平稳 → 有协整；残差不平稳 → 无协整（伪回归）
  - 为什么A"一定存在协整"错：两个序列可能各自有趋势但完全无关
  - 为什么C"误差修正模型一定成立"错：必须先确认协整，才能建ECM

### 单选Q3：当随机误差项存在自相关时，进行单位根检验用？

- **答案**：B. ADF检验
- **知识点**：
  - DF假设残差无自相关，如果残差有自相关 → DF统计量不准
  - ADF在DF方程基础上加入ΔY的滞后项 → 吸收掉残差的自相关 → 统计量才准
  - ⚠ 不选A（DF）：DF处理不了残差有自相关的情况
  - ⚠ 不选C（EG）：EG是协整检验，不是平稳性检验
  - ⚠ 不选D（DW）：DW是检验自相关的，不是做平稳性检验的

### 单选Q4：有关EG检验的正确说法？

- **答案**：A. 拒绝零假设说明被检验变量之间存在协整关系
- **知识点**：
  - EG检验的H₀是"无协整"，拒绝H₀ = 有协整
  - B"接受零假设说明存在协整"——完全反了
  - C"拒绝零假设说明不存在协整"——也反了
  - D"接受零假设说明不存在协整"——意思对但表述不规范，应说"不拒绝"而非"接受"，且和A重复

## 三、多项选择题

### 多选Q1：平稳性检验的方法有？

- **答案**：ABCD全选（散点图、自相关函数检验、单位根检验、ADF检验）
- **知识点**：
  - 和填空Q1考的是同一个知识点，三种方法：图法、ACF法、单位根法
  - ⚠ C"单位根检验"和D"ADF检验"其实都属于单位根检验（DF和ADF），但题目分开列了，都要选

### 多选Q2：当时间序列是非平稳的时候？

- **答案**：ABCD全选
- **知识点**：
  - 这道题本质在考"宽平稳三条件"——非平稳就是三条件全部或部分被打破：
    - A. 均值函数不再是常数 ✓（均值随时间变，有趋势）
    - B. 方差函数不再是常数 ✓（波动忽大忽小）
    - C. 自协方差函数不再是常数 ✓（相关结构随时间改变）
    - D. 统计规律随时间的位移而发生变化 ✓（ABC的总结性表述）
  - 宽平稳三条件（反过来记更好用）：①均值恒定 ②方差恒定 ③自协方差只取决于滞后阶数k，不取决于时点t

### 多选Q3：随机游走序列是？

- **答案**：BD（非平稳序列 + 统计规律随时间位移变化）
- **知识点**：
  - 随机游走：Yₜ = Yₜ₋₁ + εₜ，每期冲击永久累积，方差Var(Yₜ) = tσ² 随时间无限增大
  - A"平稳序列"——错，随机游走是典型的非平稳
  - B"非平稳序列"——对 ✓
  - C"统计规律不随时间变化"——错，方差在涨，统计规律一直在变
  - D"统计规律随时间变化"——对 ✓
  - 与白噪声的关系：εₜ是白噪声 → 白噪声是随机游走的驱动项；对随机游走做一阶差分就得到白噪声

### 多选Q4：下面可以做协整性检验的有？

- **答案**：CD（EG检验 + Johansen检验）
- **知识点**：
  - 协整检验：检验多个非平稳序列之间有没有长期均衡关系
    - C. EG检验：一阶协整（Y和单个X）✓
    - D. Johansen检验：多阶协整（多个变量）✓
  - A的DF和B的ADF是**平稳性检验**（检验单个序列平不平稳），不是协整检验
  - ⚠ 这是高频易错点：平稳性检验（DF/ADF）和协整检验（EG/JJ）是两回事
    - 平稳性 → 针对单个序列
    - 协整 → 针对多个序列之间的关系

### 多选Q5：有关DF检验的正确说法？

- **答案**：BC
- **知识点**：
  - A. "零假设是被检验时间序列平稳" ✗ → 反了，H₀是非平稳
  - B. "零假设是被检验时间序列非平稳" ✓
  - C. "DF检验是单侧检验" ✓
  - D. "DF检验是双侧检验" ✗
  - 为什么是单侧：
    - DF检验的假设：H₀: φ = 0（非平稳），H₁: φ < 0（平稳/衰减）
    - H₁只检验φ<0这一个方向（衰减），不检验φ>0（爆炸）
    - 所以只看分布的左侧，是单侧检验
  - 对比t检验：t检验的H₁是β≠0（两个方向都看），所以是双侧

## 四、计算题

### 计算Q1：联立方程组——判断内生/外生/先决变量 + 简化式

**题干**：下列为一完备的联立方程计量经济学模型：

- Yₜ = β₀ + β₁Mₜ + γ₁Cₜ + γ₂Iₜ + μ₁ₜ
- Mₜ = α₀ + α₁Yₜ + γ₃Pₜ + μ₂ₜ

其中M为货币供应量，Y为国内生产总值，P为价格总指数。C、I分别为消费和投资。

**(1) 指出模型的内生变量、外生变量和先决变量**

- **内生变量：Y、M**
  - 判断方法：看变量有没有"它 = 什么"的方程 → Y有（第一个方程），M有（第二个方程）→ 都在等式左边被解释过 → 内生
  - 定义：具有某种概率分布的随机变量，其参数是联立方程系统估计的元素，由系统变量决定
- **外生变量：C、I、P**
  - 判断方法：方程组里没有方程解释它们（只出现在右边，没有"C = 什么"这样的方程）→ 取值由系统外部给定
  - 定义：与扰动项不相关，Cov(Xᵢ, Uᵢ) = 0
- **先决变量：C、I、P**
  - 定义：外生变量 + 内生变量的滞后项
  - 本题无滞后项 → 先决变量 = 外生变量
- **⚠ 内生性问题**：M出现在第一个方程右边当解释变量，但M又被第二个方程里的Y决定 → Cov(M, μ₁) ≠ 0 → 直接对第一个方程做OLS，β̂₁是有偏的

**(2) 写出简化式模型，并导出结构式参数与简化式参数之间的关系**

- 简化式：把第二个方程代入第一个，消去内生变量，使Y只用外生变量表示
- 结构式是原始的联立方程（含内生变量互相影响），简化式是消元后的"纯净"形式
- 简化式参数可以由结构式参数推导出来，用于识别和估计结构参数

### 计算Q2：AR(2)平稳性判断

**题干**：有如下AR(2)随机过程：Xₜ = 0.1Xₜ₋₁ + 0.06Xₜ₋₂ + εₜ，该过程是否是平稳过程？

**解题步骤**：

1. **写特征方程**：1 − 0.1z − 0.06z² = 0
2. **分解因式**：(1 + 0.2z)(1 − 0.3z) = 0
3. **求根**：z₁ = −5，z₂ = 10/3
4. **判断**：|z₁| = 5 > 1，|z₂| = 10/3 ≈ 3.33 > 1 → 所有根的模都大于1 → 根都在单位圆外

**结论**：特征方程的根都在单位圆外，所以该过程是**平稳**的。

**知识点**：

- AR(p)平稳条件：特征方程所有根的模 > 1（在单位圆外）
- 如果有根 = 1 → 单位根 → 非平稳（这就是"单位根"名字的来历）
- 如果有根的模 < 1 → 序列会爆炸 → 也非平稳
- ⚠ 这里的z是滞后算子的根，z = 1/φ；|φ| < 1（系数绝对值小于1，衰减）等价于 |z| > 1（根在圆外）

### 计算Q3：MA(3)求自协方差和自相关函数

**题干**：求MA(3)模型 Yₜ = 1 + μₜ + 0.8μₜ₋₁ − 0.5μₜ₋₂ + 0.3μₜ₋₃ 的自协方差和自相关函数。

**读出参数**：

- MA模型标准形式：Yₜ = μₜ − θ₁μₜ₋₁ − θ₂μₜ₋₂ − θ₃μₜ₋₃
- 对比题目：0.8对应−θ₁ → θ₁ = −0.8；−0.5对应−θ₂ → θ₂ = 0.5；0.3对应−θ₃ → θ₃ = −0.3
- ⚠ 符号要看清楚，不同教材MA模型的正负号写法不同

**自协方差（直接套公式）**：

- γ₀ = (1 + θ₁² + θ₂² + θ₃²)σ²μ = (1 + 0.64 + 0.25 + 0.09)σ²μ = **1.98σ²μ**
- γ₁ = (−θ₁ + θ₁θ₂ + θ₂θ₃)σ²μ = (0.8 + (−0.8)(0.5) + (0.5)(−0.3))σ²μ = **0.25σ²μ**
- γ₂ = (−θ₂ + θ₁θ₃)σ²μ = (−0.5 + (−0.8)(−0.3))σ²μ = **−0.26σ²μ**
- γ₃ = −θ₃σ²μ = −(−0.3)σ²μ = **0.3σ²μ**
- γₖ = 0（k > 3）← **MA(q)的自协方差在k > q后截尾**

**自相关函数**（ρₖ = γₖ / γ₀）：

- ρ₀ = 1
- ρ₁ = 0.25 / 1.98 = **0.126**
- ρ₂ = −0.26 / 1.98 = **−0.131**
- ρ₃ = 0.3 / 1.98 = **0.152**

**知识点**：

- MA(q)核心性质：ACF在q阶后截尾（γₖ = 0 when k > q）
- 这就是为什么"ACF截尾 → 选MA模型"的数学原因
- 自协方差公式需要背，考试直接套数字算
- ⚠ 自相关函数ρₖ = γₖ/γ₀，就是把自协方差归一化（都除以γ₀）

## 试卷2总结

### 本张试卷的核心知识脉络

这张试卷集中考**时间序列**板块，知识点串联起来是一条完整的链：

1. 拿到序列 → **平稳性检验**（填空Q1、Q2、Q3，多选Q1、Q2，单选Q3，多选Q5）
2. 不平稳 → **差分**变平稳（单选Q1：差几次 = 几阶单整）
3. 两个I(1)序列 → **协整检验**（填空Q4、Q6，单选Q2、Q4，多选Q4）
4. 有协整 → **ECM**描述短期调整（填空Q8）
5. 无协整 → **伪回归**（填空Q7）
6. 平稳后 → 建AR/MA模型 + 判断平稳性（计算Q2：AR特征根判断）
7. MA模型 → 求自协方差和ACF（计算Q3）
8. 联立方程 → 内生/外生/先决变量（计算Q1）

### 本张试卷的易错点

- 两个I(1)变量 ≠ 一定有协整（单选Q2），还需对残差做ADF
- DF/ADF是平稳性检验，EG/JJ是协整检验——两者不能混（多选Q4）
- DF检验H₀是"非平稳"，拒绝才是好消息（填空Q5，方向反直觉）
- DF检验是单侧不是双侧（多选Q5）
- EG检验拒绝H₀ = 有协整，不是无协整（填空Q4，单选Q4）
- AR平稳条件是特征根在单位圆**外**（|z| > 1），不是圆内
- MA模型θ的符号要看清教材写法，正负号对应可能不同（计算Q3）
