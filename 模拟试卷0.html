编号假定公式含义

Q2 模型拟合效果的评判

（1）拟合优度

（2）是否有异方差

（3）是否有结构突变

（4）约束条件（受约束F检验）

答案要点

（1）拟合优度

R² = ESS/TSS = 1 − RSS/TSS（越接近1越好）
调整R²：惩罚多余变量，加无用变量可能下降
AIC/BIC：越小越好，BIC惩罚更重，不一致时优先BIC

（2）异方差检验

White检验最通用：残差²对X及交叉项/平方项回归 → nR²~χ²，H₀:不存在异方差。读表看Obs*R-squared行。

（3）结构突变：Chow检验

F = [(RSS_pooled − RSS₁ − RSS₂)/(k+1)] / [(RSS₁+RSS₂)/(n−2(k+1))]
H₀：两段参数相同 → 拒绝H₀ → 存在结构突变 → 分段建模或加时间虚拟变量

（4）受约束F检验

F = [(RSS_R − RSS_UR)/q] / [RSS_UR/(n−k−1)]
q = 约束个数；RSS_R ≥ RSS_UR（加约束只会使残差变大）
H₀：约束成立 → 拒绝则约束不成立

Chow检验是受约束F检验的特例（约束 = "两段系数完全相同"）

Q1 模型识别：写出以下模型的类型和参数
(1) Yₜ = c + α₁Yₜ₋₁ + α₂Yₜ₋₂ + εₜ
(2) Yₜ = μ + θ₁εₜ₋₁ + θ₂εₜ₋₂ + θ₃εₜ₋₃
(3) εₜ² = α₀ + α₁ε²ₜ₋₁ + α₂ε²ₜ₋₂ + α₃ε²ₜ₋₃ + α₄ε²ₜ₋₄
(4) σ²ₜ = ω + Σᵢ₌₁⁴ αᵢε²ₜ₋ᵢ + Σⱼ₌₁⁶ βⱼσ²ₜ₋ⱼ
(5) Yₜ = c + α₁Yₜ₋₁ + α₂Yₜ₋₂ + εₜ + θ₁εₜ₋₁ + θ₂εₜ₋₂ + θ₃εₜ₋₃

追问：我国股票适合用什么模型解释？

答案
编号模型识别依据参数
(1)AR(2)只有Y的滞后项，2个p = 2
(2)MA(3)只有ε的滞后项，3个q = 3
(3)ARCH(4)方差方程只有ε²项，4个q = 4
(4)GARCH(6,4)方差方程有ε²项(4个) + σ²项(6个)p=6, q=4
(5)ARMA(2,3)既有Y滞后(2个)又有ε滞后(3个)p=2, q=3

识别口诀

  • 只有Y的滞后项 → AR，数滞后项个数 = p
  • 只有ε的滞后项 → MA，数滞后项个数 = q
  • 两者都有 → ARMA
  • 方差方程只有ε²项 → ARCH
  • 方差方程有ε²项 + σ²项 → GARCH，p=σ²项个数，q=ε²项个数

追问：股票适合什么模型？

ARMA(p,q)。其中AR部分反映股价变化的动态惯性——过去的价格对当前有持续影响；MA部分反映冲击因素的阶段性影响——过去的预测误差对当前值的影响。

如果波动率也需要建模（波动聚集现象），则在ARMA基础上加GARCH。

Q2 简述如何判断白噪声，并举例应用
答案要点

白噪声定义

三个条件：
① E(εₜ) = μ（均值恒定）
② Var(εₜ) = σ²（方差恒定）
③ Cov(εₜ, εₛ) = 0, t≠s（完全无自相关）

白噪声 = 宽平稳 + 无自相关，比宽平稳更严格

判断方法（三种）

  1. ACF图观察法：画残差的ACF图，若绑大多数都落在置信区间边界内 → 白噪声
  2. Box-Pierce检验：Q_BP = n · Σρ̂ₖ² ~ χ²(m)，H₀:是白噪声
  3. Ljung-Box检验 ★（BP改进版）：Q_LB = n(n+2) · Σ[ρ̂ₖ²/(n−k)] ~ χ²(m)，H₀:是白噪声

⚠ H₀是"好消息"（是白噪声），不拒绝H₀才是好结果——和大多数检验方向相反！

三大应用场景

对什么做问的问题是白噪声不是白噪声
① 对平稳序列能预测吗？完全随机，不值得建模有自相关，可以建模 ✓
② 对模型残差拟合好吗？信息提取干净 ✓还有信息没提取，调整模型
③ 对ε²波动恒定吗？无ARCH效应 ✓有ARCH效应，需建GARCH
Q3 如何进行平稳性检验？
答案要点

三种方法

  1. 时间路径图：直接画图看趋势和方差变化（直观但不严格）
  2. ACF图：平稳→ACF快速衰减；非平稳→ACF缓慢衰减
  3. 单位根检验（ADF）★：
    • 写出特征方程，看有没有模长=1的根（单位根）
    • 有单位根 → 非平稳；所有根在单位圆外（模>1）→ 平稳
    • H₀：存在单位根（非平稳），拒绝H₀ = 平稳 = 好消息
    • DF针对AR(1)残差无自相关；ADF拓展到高阶/残差有自相关

⚠ 拒绝H₀才是好消息——方向反直觉！ADF使用DF分布临界值，不是标准t分布

Q4 如何进行模型识别？
答案要点

方法一：ACF/PACF图

ACFPACF选什么模型
拖尾截尾（p阶后=0）AR(p)
截尾（q阶后=0）拖尾MA(q)
拖尾拖尾ARMA(p,q)

速记：AR→PACF截，MA→ACF截，ARMA→全拖

AIC = ln(σ̂²) + 2k/n     BIC = ln(σ̂²) + k·ln(n)/n
越小越好。在候选模型中选AIC/BIC最小的。不一致时优先BIC。

Q2 什么是固定效应模型和随机效应模型？如何进行模型识别？
答案要点

面板数据模型：Yᵢₜ = α₀ + β₁Xᵢₜ + uᵢ + εᵢₜ，其中 uᵢ 是个体效应（个体之间的差异）。

→ 核心问题：uᵢ 和 Xᵢₜ 有没有相关？

固定效应模型（FE）随机效应模型（RE）
核心假设uᵢ 与 Xᵢₜ 相关（有内生性）uᵢ 与 Xᵢₜ 不相关（无内生性）
处理方式uᵢ 看作截距的一部分
（每个个体有不同截距）		uᵢ 看作扰动项的一部分
（复合扰动 = uᵢ + εᵢₜ）
估计方法组内估计（去均值消除uᵢ）GLS
局限不随时间变化的变量系数无法估计若uᵢ与X相关则有偏

模型选择：三步决策

第一步 F检验：混合模型 vs 固定效应
→ H₀: 混合模型更优（无个体效应）→ 拒绝则有个体效应

第二步 BP检验：混合模型 vs 随机效应
→ H₀: 无异方差（用混合）→ 拒绝则用随机效应

第三步 Hausman检验 ★（最重要）：固定效应 vs 随机效应
→ H₀: 无内生性（FE和RE都一致，RE更有效 → 选RE）
→ 拒绝H₀: 有内生性（只有FE一致 → 选FE）

实务中一般无脑选固定效应也OK。

Q3 面板数据模型中，工具变量法有何应用？工具变量的选取有什么要求？
答案要点

应用场景

面板数据中如果存在内生性（Hausman检验拒绝H₀），除了用固定效应外，还可以引入工具变量（IV），通过2SLS消除内生性。

Hausman检验本身就是在比较OLS（或RE）和IV的结果——差距显著说明有内生性，应使用IV或FE。

工具变量Z的选取要求（三条）

  1. 相关性：Cov(Z, X) ≠ 0 → Z和内生解释变量X高度相关（可统计检验）
  2. 外生性：Cov(Z, U) = 0 → Z和扰动项U不相关（依赖经济理论，不可直接检验）
  3. 排斥性：IV不应作为解释变量直接出现在方程中（Z对Y的影响只通过X传递）

⚠ 只满足相关性不够！2015年真题考过。相关性+外生性缺一不可。

Q4 面板数据模型随机误差项的假设有哪些？
答案要点
  1. 每个截面的误差项零均值：E(εᵢₜ) = 0
  2. 每个个体对应的误差项同方差：Var(εᵢₜ) = σ²（同一个体不同时期方差相同）
  3. 不同截面间残差不相关：Cov(εᵢₜ, εⱼₜ) = 0, i≠j（个体i和个体j的残差不相关）
  4. 不同截面间具有同方差：不同个体的误差方差相同

对比OLS的古典假定：面板数据的假定多了一个"截面维度"——不仅要求同一个体的时序残差满足假定，还要求不同个体之间的残差也满足。

如果违反了截面间同方差 → 截面异方差；违反了截面间不相关 → 截面相关。都需要用更稳健的估计方法。
