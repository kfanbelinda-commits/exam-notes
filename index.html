1. 从问题出发：计量经济学在干什么？

生活中我们想预测一件事——比如收入增加了，储蓄会增加多少？

为了回答这个问题，我们把"收入影响储蓄"写成数学关系：

总体回归方程：Y = β₀ + β₁X + U
Y = 储蓄（想预测的） |  X = 收入（原因） |  β₁ = 收入每增1元，储蓄平均增多少  |  U = 其他乱七八糟的影响（看不见）

→ 但问题来了：β₁ 到底是多少？

β₁ 是总体参数——我们永远不知道它的真实值（除非调查全国每一个人）。我们只有一份样本数据，只能用样本去估计它。估计出来的值叫 β̂₁（帽子 = "估计值"）。

→ 那怎么估计？总得有个标准吧。

标准就是：让估计出来的回归线尽可能贴近数据。怎么衡量"贴近"？看每个数据点到回归线的竖直距离（残差），让这些距离的平方和最小。

为什么用平方和？因为残差有正（点在线上方）有负（点在线下方），直接加会互相抵消。平方后全部变正，大的偏离被加倍惩罚。

图1：OLS 找到一条线，使所有红色虚线（残差）的平方和最小
最小化目标：min Σ eᵢ² = min Σ [Yᵢ − (β̂₀ + β̂₁Xᵢ)]²

怎么求解：对 β̂ 求偏导，令偏导 = 0
→ 正规方程 X'Xβ̂ = X'Y
→ 解出 β̂ = (X'X)⁻¹X'Y

⚠ 2026年已考：写出 β̂ 的推导过程 + 证明无偏性。这是必须闭卷默写的公式。

→ β̂ 算出来了，但还有两个问题要回答：

  1. β̂ 准不准？（是不是在"瞄准"真值β？）→ 这取决于古典假定（第2节）
  2. 模型好不好？（解释了多少Y的变动？）→ 这要看 R²（下面马上讲）

1.1 残差 vs 误差——先分清这两个「差」

在回答"准不准"之前，先分清两个容易混的概念：

误差 U（Error）
总体角度，不可观测
Uᵢ = Yᵢ − (β₀ + β₁Xᵢ)
真实值偏离真实回归线多少
永远看不见（因为β₀, β₁未知）
残差 e（Residual）
样本角度，可以算出
eᵢ = Yᵢ − Ŷᵢ = Yᵢ − (β̂₀ + β̂₁Xᵢ)
真实值偏离估计的回归线多少
β̂已知，所以能算出来

类比：误差U是"上帝知道的真实偏离"，残差e是"我们能算出的对偏离的估计"。残差是误差的估计值。

→ 既然误差U看不见，那我们怎么判断模型好不好？靠残差。具体来说，靠 TSS 分解和 R²。

1.2 模型好不好？—— TSS 分解与 R²

对散点图上每一个数据点 i，都可以把Y偏离均值的总距离（离差）拆成两部分：

图2：对每个点 i，总变动（TSS）= 模型解释的（ESS）+ 模型没解释的（RSS）
对所有点取平方和 → TSS = ESS + RSS

TSS = Σ(Yᵢ − Ȳ)² — Y 的总变动
ESS = Σ(Ŷᵢ − Ȳ)² — 模型解释了的部分（回归线偏离均值线多远）
RSS = Σ(Yᵢ − Ŷᵢ)² — 模型没解释的部分（数据点偏离回归线多远）

→ 那"模型好不好"就可以量化了：ESS占TSS的比例越大，模型越好。这个比例就是 R²。

R² = ESS / TSS = 1 − RSS / TSS

R² = 1：RSS = 0，所有点在回归线上 → 完美拟合（ESS = TSS，模型解释了全部变动）
R² = 0：ESS = 0，Ŷᵢ = Ȳ，回归线退化为水平均值线 → 模型什么都没解释

调整 R²：R² 有个缺陷——加变量必然增大（哪怕加的是无用变量）。调整R² 会惩罚无用变量，加错变量可能反而下降。
AIC / BIC：另一种衡量模型好坏的指标，越小越好。BIC惩罚更重，AIC与BIC不一致时优先BIC。

→ 现在我们知道了β̂怎么算（OLS）、模型好不好怎么看（R²）。但最关键的问题还没答：β̂ 到底准不准？

2. β̂ 准不准？—— 古典假定与高斯-马尔科夫定理

答案是：如果古典假定全部成立，β̂就是最好的估计。"最好"的意思是：在所有线性无偏估计量中，β̂的方差最小（波动最小，最精确）。

→ 那古典假定具体是什么？一共六条：

2.1 六条古典假定（简答必背）

编号假定公式直觉理解
①X 非随机X 是确定性的完全随机的东西无法预测——你拿噪声当原因，能预测出什么？
②零均值E(Uᵢ) = 0偏离有正有负，但平均为零——不系统性地偏向某一边
③同方差Var(Uᵢ) = σ²偏离的波动前后一致——不能收入低时波动小、收入高时波动大
④无自相关Cov(Uᵢ,Uⱼ)=0前一期残差不影响后一期——残差不能"手拉手"往一个方向跑
⑤正态分布Uᵢ ~ N(0,σ²)大部分偏离集中在零附近，越大的偏离越少（钟形分布）
⑥无完全共线性X'X 可逆解释变量之间不能完全线性相关——否则分不清谁在起作用

⚠ ③同方差 + ④无自相关合称"球型扰动假设"，经常一起考，缺一不可（2010年真题考过）

→ 这六条全部成立时，就有了下面这个定理：

2.2 高斯-马尔科夫定理

满足六条假定 → β̂ 具有 BLUE 性质：

  • Best = 最优的（方差最小）
  • Linear = β̂ 是 Y 的线性函数
  • Unbiased = 无偏：E[β̂] = β（反复抽样，β̂的平均值等于真值）
  • Estimator = 估计量

一句话：在所有"Y的线性函数+无偏"的估计量中，OLS的β̂方差最小。没有比它更精确的了。

无偏性证明（2026已考）

β̂ = (X'X)⁻¹X'Y
→ 代入 Y = Xβ + U
→ β̂ = (X'X)⁻¹X'(Xβ + U) = β + (X'X)⁻¹X'U
→ 取期望：E[β̂|X] = β + (X'X)⁻¹X' · E[U|X]
→ 关键一步：假定②要求 E[U|X] = 0
→ 所以 E[β̂|X] = β   ✓（无偏）

大样本额外获得的性质：一致性 — n→∞ 时 β̂→β（样本量越大，估计越准）

→ 但现实中，六条假定几乎不可能全部满足。违反了某条假定 → BLUE 就不再成立 → β̂ 可能不准了。怎么办？

→ 分两步：先检验（有没有违反？）→ 再修正（违反了怎么补救？）

3. 假定被违反了——检验、后果、修正

OLS估计完β̂之后，要做一轮"体检"：逐项检查六条假定有没有被违反。发现问题 → 修正 → 修正后才能放心下结论。

💡 下面按假定③④⑥+外生性展开（假定①②⑤在实践中较少单独检验，但简答题要能写出来）。

3.1 假定③被违反 → 异方差

什么是异方差？

假定③要求残差方差恒定（Var(Uᵢ) = σ²）。如果方差随X变化——比如收入越高，消费的波动越大——就叫异方差。

图3：同方差 vs 异方差的残差散点图对比

→ 怎么判断有没有异方差？除了直接看图，还有4种统计检验：

异方差检验（5种方法）

#方法做法特点
①散点图画残差²对X的图，看有没有喇叭口最直观，但不严格
②GQ检验按X排序分两段，比较两段RSS → F检验适用递增/递减型
⚠ GQ是异方差检验，不是自相关！
③White检验 ★残差²对X及其交叉项、平方项做辅助回归 → nR²~χ²最通用，不需假设形式
④Glejser检验|残差|对X回归可推断异方差函数形式
⑤BP检验残差²对X回归 → nR²~χ²比White简洁（不加交叉项）

H₀：不存在异方差（同方差） → 拒绝H₀ = 有异方差 = 需要修正

→ White检验为什么这么做？思路是：

如果同方差成立，残差²应该是随机的，跟X没关系。所以把残差²当Y，X及其平方项、交叉项当解释变量做辅助回归。如果这个辅助回归的R²很大，说明残差²能被X解释——方差跟着X变——异方差。

⚠ 读表时看 Obs*R-squared 那行的 Probability，不看 F-statistic 那行（避免临界值附近两行结论不一致）

→ 如果发现了异方差，有什么后果？怎么修正？

后果

OLS 仍然无偏且一致（β̂还是在瞄准真值），但不再最小方差（BLUE被打破）。标准误公式算错了 → t/F检验结论不可靠。

类比：你的枪瞄得很准（无偏），但枪管松了导致散布变大（方差增大）。打10发平均命中，但单发可能偏很远。

修正

  • WLS（加权最小二乘）：给方差大的观测降低权重，恢复同方差
  • 怀特稳健标准误：不改β̂（还是那个数），只修正标准误的计算公式 → t值变了 → 显著性结论可能变

3.2 假定④被违反 → 自相关

什么是自相关？

假定④要求不同观测的误差互不相关。如果前一期残差影响后一期——残差"手拉手"往一个方向跑——就叫自相关。

时间序列数据尤其常见：今天股价涨了，明天大概率还涨（正自相关）；或者今天涨明天跌（负自相关）。

→ 有两种检验方法：

自相关检验（2种）

① DW检验（Durbin-Watson）— 只能检验一阶自相关

DW ≈ 2(1 − ρ̂)，ρ̂ = 相邻残差的相关系数
直接代入：ρ̂=1 → DW=0（完全正自相关）| ρ̂=0 → DW=2（无自相关✓）| ρ̂=−1 → DW=4（完全负自相关）
图4：DW检验的区间判断

限制：只能检验一阶自相关；要求模型含截距项、不含被解释变量滞后项

② BG-LM检验 ★ 更通用— 可以检验高阶自相关

做法：残差对自身滞后项做自回归 → nR² ~ χ²

H₀：不存在自相关 → 读表看 Obs*R-squared 行：p < 5% → 有自相关

⚠ DW是自相关检验，不是异方差检验（GQ才是异方差）——选择题高频考点！

后果

和异方差一样：OLS仍无偏一致，但标准误有偏，t/F不可靠。

修正

  • 广义差分：对Y和X同时做 (Yₜ − ρ̂Yₜ₋₁) 变换
  • Cochrane-Orcutt迭代法：先估 ρ̂ ≈ 1 − DW/2，再做广义差分

⚠ ρ̂ ≈ 1 − DW/2（不是DW本身！）考过：DW=0.52 → ρ̂ = 1−0.52/2 = 0.74，不是0.52

💡 金融计量学恰恰要利用自相关来建模预测（→ 第二条线），而不只是"修正"它。

3.3 假定⑥被违反 → 多重共线性

什么是多重共线性？

解释变量之间高度线性相关。比如"身高"和"臂展"同时放进回归——两者几乎成正比，模型分不清谁在起作用。

→ 怎么判断？（4种方法，不是假设检验的形式，没有H₀）

#方法判断标准
①简单相关系数|Rᵢⱼ| ≈ 1 → 两个变量高度相关
②VIF（方差膨胀因子）VIF = 1/(1−R²ⱼ) > 5 或 > 10 → 严重
③F大R²大但t小联合显著但各自不显著 → 典型表现
④条件指数CICI > 20 → 有共线性

后果（讲师推导链——简答必背）

完全共线性：X'X不可逆 → 无法求出β̂（直接炸了）

不完全共线性推导链：
解释变量间相关性高 → 二阶矩增大 → VIF↑ → Var(β̂)↑
→ ① 不再具有最小方差性（BLUE被打破）
→ ② t = β̂/se(β̂) ↓（分母se变大）→ t检验失效
→ ③ 置信区间变宽 → 预测精度下降

⚠ 关键区别：多重共线性时 F检验和R²不受影响（只有t检验受影响）——这是和异方差/自相关不同的地方

修正

剔除不重要变量 / 增加样本量 / 逐步回归（AIC/BIC选变量）/ 岭回归

3.4 外生性假定被违反 → 内生性（最严重！）

什么是内生性？

解释变量X和扰动项U相关：Cov(X, U) ≠ 0。

前面三种违反（异方差/自相关/共线性）都只影响"精确度"，β̂还是在瞄准真值。但内生性直接让β̂偏离靶心——不仅不够精确，方向都偏了，而且样本量再大也救不回来。

三种来源

  1. 遗漏变量（最常见）：重要变量被遗漏进入U，且与X相关。例：研究教育对收入的影响，但"能力"这个变量无法观测，被遗漏进入U，而能力和教育年限相关 → Cov(教育, U) ≠ 0
  2. 测量误差：X的观测值含误差，误差进入U且与X相关
  3. 联立性/反向因果：X影响Y的同时Y也影响X（如价格与需求互相决定）→ 联立方程问题

→ 怎么检验有没有内生性？

Hausman检验

思路：如果没有内生性，OLS就够好了，工具变量法（IV）和OLS结果应该差不多。如果结果差很大，说明OLS偏了 → 有内生性。

H₀：不存在内生性

拒绝H₀：有内生性 → OLS有偏且不一致！
不拒绝H₀：外生OK ✓，用OLS就行

后果

OLS 有偏且不一致——唯一破坏无偏性和一致性的违反！

修正：工具变量法（IV）/ 两阶段最小二乘（2SLS）

核心思路：X"脏了"（和U相关），找一个"干净的替身"Z。Z必须同时满足：

  1. 相关性：Cov(Z, X) ≠ 0 → Z和X高度相关
  2. 外生性：Cov(Z, U) = 0 → Z和U不相关

⚠ 只满足相关性不够！2015年真题考过。

2SLS 两步操作：
第一步：用Z回归X → 得到拟合值X̂（X̂只保留了X中被Z解释的"干净"部分）
第二步：用X̂替代原来的X，对Y做OLS → 得到一致的β̂

代价：IV估计的方差比OLS大——用效率换一致性。

假设检验四步流程（万能模板）

① 列假设：写出 H₀ 和 H₁
② 构造统计量：代入数据算出 t / F / χ² 的数值
③ 查临界值：根据α和自由度，查分布表得到门槛值
④ 比较判断：|统计量| vs 临界值 → 套口诀

"率" = p值 vs α → p > α = "率大" → 不拒绝

⚠ F检验是单侧 → 直接用 F₀.₀₅
⚠ 但两者等价：t₀.₀₂₅² = F₀.₀₅（一元回归时）

从第一条线到第二条线：为什么需要时间序列？

在第一条线里，我们学了OLS回归的基础。OLS有一条假定叫"无自相关"——要求残差前后期互不影响。如果违反了，我们把它当作"问题"去修正。

→ 但金融数据恰恰就是自相关的！

股价今天涨了，明天大概率还涨；今天跌了，明天大概率还跌。这种"惯性"就是自相关。如果我们只是把自相关当"问题"去消除，等于扔掉了最有价值的信息。

时间序列分析的核心思想：不是消除自相关，而是利用自相关来建模预测。把序列拆成两部分：

确定部分（可预测）
序列有自相关 + 平稳
→ 用AR/MA/ARMA建模
→ 历史信息可以预测未来
随机部分（不可预测）
残差无自相关 + 平稳
→ 希望是白噪声
→ 说明确定部分已提取干净

类比：序列像一首歌，"确定部分"是旋律（有规律可循），"随机部分"是底噪（完全随机）。建模就是把旋律识别出来，剩下的纯噪音说明旋律已经提取完了。

→ 整个时间序列建模就是一个固定流程，四步走完：

下面逐步展开。每一步都是：为什么要做这一步 → 怎么做 → 结果怎么判断 → 不通过怎么办。

第一步：序列平不平稳？

1.1 为什么第一步就要看平稳性？

平稳的意思是：序列的统计特征（均值、方差、自协方差）不随时间变化。这意味着过去的规律未来还会继续——这是预测的前提。

如果序列不平稳（比如股价一直在涨），过去的规律不一定适用于未来。更严重的问题是：两个不平稳序列直接回归会产生伪回归——R²虚高但无真实关系。

→ 所以拿到数据第一件事：检查平不平稳。

1.2 平稳性的三个条件（宽平稳）

① 均值恒定：E(Yₜ) = μ，不随时间变（没有趋势）
② 方差恒定：Var(Yₜ) = σ²，不随时间变（波动幅度稳定）
③ 自协方差只取决于滞后阶数k，不取决于时点t：Cov(Yₜ, Yₜ₋ₖ) = γₖ

直觉：平稳序列像围绕一条水平线上下波动，波动幅度前后差不多。非平稳序列像一直往上走或波动越来越大。

图1：平稳序列 vs 非平稳序列

三种方法，从直观到严格：

②ACF图画自相关函数图平稳→ACF快速衰减；非平稳→ACF缓慢衰减（拖很长的尾巴）
③ADF检验 ★统计假设检验最严格，考试必用

→ 考试用的是ADF检验，重点展开：

1.4 ADF检验（核心）

先从最简单的情况说起：AR(1)

AR(1)模型：Yₜ = φ·Yₜ₋₁ + εₜ。展开后，每期历史冲击 εₖ 传到今天被乘以 φᵏ 作为权重：

|φ| < 1
φᵏ 随k增大快速衰减
历史冲击影响消失
→ 平稳
φ = 1
φᵏ 永远等于1
每期冲击100%保留，永不衰减
→ Var(Yₜ)=tσ² 随时间增大
→ 非平稳（随机游走）
|φ| > 1
φᵏ 越来越大
冲击被放大 → 爆炸
→ 更不平稳

φ=1就是"单位根"——平稳与不平稳的分界线。过去的冲击永不衰减，像喝醉酒的人越走越偏。

高阶AR(p)：用特征方程

AR(1)只有一个系数φ，直接看就行。但AR(p)有多个系数（φ₁, φ₂, …, φₚ），怎么判断？

→ 引入滞后算子L（或B），把所有滞后项整理成一个多项式方程：

滞后算子：L·Yₜ = Yₜ₋₁，L²·Yₜ = Yₜ₋₂，…

AR(p)整理后：(1 − φ₁L − φ₂L² − … − φₚLᵖ) = 0

平稳条件：这个多项式方程的所有根的模 > 1（都在单位圆外）
有根的模 = 1 → 有单位根 → 非平稳
有根的模 < 1 → 爆炸 → 非平稳

⚠ 根和系数是倒数关系：根 L = 1/φ。|φ|<1（衰减→平稳）等价于 |L|>1（根在圆外）。"根在圆外"和"系数小于1"说的是同一件事。

不平稳怎么办？→ 差分

对序列做差分：ΔYₜ = Yₜ − Yₜ₋₁。差分后再做ADF检验。差几次变平稳就叫几阶单整 I(d)。

一次差分变平稳 → I(1)（一阶单整）
两次差分变平稳 → I(2)（二阶单整）
本身就平稳 → I(0)

实际中常用对数差分：先取对数 ln(Pₜ)，再差分 ln(Pₜ) − ln(Pₜ₋₁)
对数差分 ≈ 收益率（当变化率很小时，ln(1+x) ≈ x）
例：股价序列（非平稳）→ 取对数 → 差分 → 收益率序列（平稳）→ 可以建模了

→ 序列平稳了（自己的或差分后的），接下来要确定：它是什么类型的自相关？

第二步：序列是什么类型的自相关？

2.1 为什么要识别类型？

平稳序列有自相关性（否则就是白噪声，不值得建模）。但自相关有不同形式——"用过去的自己预测自己"和"用过去的误差预测自己"是两种不同的机制，对应不同的模型。选错了模型，预测就不准。

→ 怎么区分？用ACF和PACF两个工具。

2.2 ACF和PACF是什么？

ACF（自相关函数）= 总相关
把序列往后错开k期，看错开前后两段有多像
包含直接影响 + 间接传递（全部路径的影响都算在里面）
k越大（错开越远）→ 相关性越弱 → "近大远小"
PACF（偏自相关函数）= 直接相关
只看 Yₜ 和 Yₜ₋ₖ 两个点之间的净关系
把中间时期的传递效应扣掉了
类比求偏导：其他变量按住不动，只看这一个

ACF = 直接影响 + 间接传递 = 总效果。PACF = 只留直接影响。所以PACF一定不会比ACF大。

公式层面的区别：ACF的分母都除以γ₀（Yₜ自己的方差），间隔k的间接影响混在里面；PACF的分母分别除以Yₜ和Yₜ₋ₖ各自的标准差，点到点归一化，只反映直接关系。

模型公式核心思想直觉

  • AR模型 → 需要看特征根（是否都在单位圆外）
  • ARMA → 检验其中AR部分

  • 方法一：看ACF/PACF图，几阶后进入置信区间 → 就是几阶
  • 方法二：信息准则，在候选模型中选AIC/BIC最小的
AIC = ln(σ̂²) + 2k/n
BIC = ln(σ̂²) + k·ln(n)/n

BIC惩罚更重（n>7时 ln(n)>2），倾向更简洁的模型
AIC与BIC不一致时，优先参考BIC（避免过拟合）

第三步：残差是白噪声吗？（模型拟合好不好？）

3.1 为什么要检验残差？

建完模型后提取残差 ε̂ₜ = Yₜ − Ŷₜ。如果模型把"确定部分"（自相关结构）都提取干净了，剩下的残差应该是纯随机的——也就是白噪声。

如果残差不是白噪声（还有自相关），说明模型没提取干净，需要调整阶数重来。

3.2 白噪声三条件

① 均值恒定：E(εₜ) = μ（通常为0）
② 方差恒定：Var(εₜ) = σ²（同方差）
③ 完全无自相关：Cov(εₜ, εₛ) = 0, t≠s

白噪声 = 宽平稳 + 无自相关。比宽平稳更严格（宽平稳允许有自相关，白噪声不允许）。

严格程度排序：严格平稳 > 白噪声 > 宽平稳

3.3 怎么检验？

① ACF图观察法：画残差的ACF图，所有ACF都在置信区间内 → 白噪声

② Box-Pierce检验（Q_BP）

统计量：Q_BP = n · Σρ̂ₖ² ~ χ²(m)

③ Ljung-Box检验（Q_LB）★ BP的改进版，更准

Q_LB = n(n+2) · Σ[ρ̂ₖ²/(n−k)] ~ χ²(m)

H₀：残差是白噪声

不拒绝H₀（p > 5%）→ 残差是白噪声 → 模型OK ✓
拒绝H₀（p < 5%）→ 残差不是白噪声 → 调整模型阶数重来

⚠ 白噪声检验的H₀方向特殊：H₀是"好消息"（是白噪声），不拒绝H₀才是好结果！和大多数检验相反。

3.4 白噪声的三大应用场景（考试爱出辨析题）

对什么做白噪声检验问的问题是白噪声不是白噪声
① 对平稳序列做序列能预测吗？完全随机，不值得建模有自相关，可以建模 ✓
② 对模型残差做模型拟合好吗？信息提取干净 ✓还有信息没提取，调整模型
③ 对ε²做波动率恒定吗？方差恒定，无ARCH效应 ✓有ARCH效应，需建GARCH

同一个工具用在不同对象上，问的问题不同，"好消息"的方向也不同。第③种就是下一步要做的。

→ 残差是白噪声了（均值方程OK），但还没完——金融数据有个特殊现象：波动聚集。

第四步：波动率恒定吗？（需不需要给方差建模？）

4.1 什么是波动聚集？

金融数据有个显著特征：大波动之后往往跟着大波动，小波动之后跟着小波动。虽然残差的均值为零、无自相关（看起来像白噪声），但方差不恒定——波动率在变化。

图3：金融序列的波动聚集——残差均值为零但方差在变

这意味着方差本身也是可以预测的——用过去的波动预测未来的波动。ARCH/GARCH就是给方差建的模型。

→ 先检验有没有这种现象（ARCH效应），有才建模。

4.2 ARCH-LM检验

检验什么：残差的平方 ε̂²ₜ 有没有自相关。如果ε̂²ₜ有自相关 = 波动率可预测 = 有ARCH效应。

做法：ε̂²ₜ 对自身滞后项做自回归 → nR² ~ χ²(q)

H₀：不存在ARCH效应（所有αᵢ = 0）

不拒绝H₀ → 无ARCH效应 → 方差恒定 → 建模完成 ✓
拒绝H₀ → 有ARCH效应 → 波动率可预测 → 需要建GARCH模型

也可以对ε̂²ₜ做LB白噪声检验：ε̂²ₜ不是白噪声 → 有ARCH效应。

4.3 ARCH模型

ARCH的思路：把时变的方差σ²ₜ也当作一个可以被预测的变量来建模——用过去的残差平方（冲击大小）预测未来的方差。

ARCH(q)方差方程：
σ²ₜ = α₀ + α₁ε²ₜ₋₁ + α₂ε²ₜ₋₂ + … + αqε²ₜ₋q

含义：今天的波动率 = 一个基准水平(α₀) + 过去q天冲击大小的加权和
昨天冲击大 → 今天方差大 → 今天大概率还是大波动

→ 但ARCH有个问题：要精确刻画波动率，可能需要很多滞后项（q很大）→ 参数太多。怎么简化？

4.4 GARCH模型（ARCH的升级版）

GARCH在ARCH基础上加了一项：方差自身的滞后值。这样只需要少量参数就能捕捉长期的波动持续性。

GARCH(p,q)方差方程：
σ²ₜ = ω + α₁ε²ₜ₋₁ + … + αqε²ₜ₋q + β₁σ²ₜ₋₁ + … + βpσ²ₜ₋p

ω = 长期均值水平
αᵢε²ₜ₋ᵢ = 过去冲击的影响（ARCH项 ≈ MA的思路）
βⱼσ²ₜ₋ⱼ = 过去波动率的影响（GARCH项 ≈ AR的思路）

本质：ARMA用在方差上——就像均值方程用ARMA，方差方程也用类似结构

ARCH vs GARCH 怎么区分？

ARCH(q)GARCH(p,q)
方差方程只有 ε²项（过去冲击）同时有 ε²项 + σ²项（过去冲击 + 过去方差）
读参数q = ε²滞后项个数p = σ²滞后项个数，q = ε²滞后项个数
判断方法方差方程里没有σ²ₜ₋ⱼ → ARCH方差方程里有σ²ₜ₋ⱼ → GARCH

记忆：GARCH(p,q) 中 p对应σ²（方差自回归），q对应ε²（冲击项）。看求和符号上界直接读出p和q。

GARCH(1,1)参数约束（考试必写）

σ²ₜ = ω + αε²ₜ₋₁ + βσ²ₜ₋₁

参数约束：
ω > 0（基准方差为正）
α ≥ 0，β ≥ 0（权重非负）
α + β < 1（弱平稳条件——严格小于！不是≤）

α + β 越接近1 → 波动持续性越强（冲击衰减越慢）

⚠ α + β < 1 是严格不等号！判断题考过"α+β≤1"——错！必须严格小于。

→ 但很多时候我们想研究的是两个变量之间的关系：收入影响消费吗？货币供应量影响GDP吗？

如果两个序列都是平稳的，直接做OLS回归就行（第一条线的内容）。但金融/宏观数据大部分是非平稳的——GDP在涨、消费在涨、股价在涨。

→ 两个非平稳序列能直接回归吗？

不能。直接回归会产生伪回归——R²虚高，t检验显著，但变量之间根本没有真实关系。原因是两个序列各自有趋势（都在涨），趋势"撞"在一起造成虚假相关。

→ 那怎么办？有没有一种方法能区分"真的有关系"和"伪回归"？

有，就是协整检验。

1. 协整——两个非平稳序列有没有长期关系？

1.1 什么是协整？

用一个生活化的比喻：

想象两个喝醉的人从酒吧出来，各自走路歪歪扭扭（非平稳）。如果他们各走各的，那两人的距离会越来越远（无协整 → 伪回归）。但如果他们手拉手——虽然各自还是歪歪扭扭的，但两人之间的距离始终保持在一个稳定范围内，偏远了会自动拉回来。这就是协整。

→ 用数学说：

收入 Y 是 I(1)（非平稳，一直在涨）
消费 X 也是 I(1)（非平稳，也在涨）

但 Y − βX（两者的线性组合）是平稳的
→ 它在一个固定均值附近上下波动，偏离了会自动回来

这个β就是长期均衡关系。例子：消费倾向 = 消费/收入，长期来看是稳定的。

单整 vs 协整：

单整：针对单个序列
差几次变平稳 = 几阶单整 I(d)
协整：针对多个序列
各自不平稳，但线性组合平稳
= 之间有长期均衡关系

1.3 EG两步法（重点展开）

思路非常简单：如果Y和X有协整，那它们的偏差（残差）应该是平稳的。所以：

第一步：对Y和X做OLS → 目的不是要回归结果，而是要拿到残差

Y = β̂₀ + β̂₁X + û
残差 û = Y − Ŷ = Y − β̂₀ − β̂₁X

û 代表什么？Y和X之间的偏差——Y偏离了长期均衡关系多少

第二步：对残差 û 做ADF单位根检验 → 看残差平不平稳

H₀：残差不平稳（无协整）

H₁：残差平稳（有协整）

残差平稳 → 拒绝H₀ → 有协整 ✓
Y和X的偏差会自动回归均值
两者存在长期均衡关系
残差不平稳 → 不拒绝H₀ → 无协整
Y和X的偏差越走越远
→ 伪回归，R²虚高无意义

2. ECM——偏离了长期均衡怎么调整回来？

2.1 为什么需要ECM？

协整告诉我们Y和X长期存在均衡关系。但短期内可能会偏离——比如某个季度消费突然偏高（超出均衡水平）。

→ 偏离了之后怎么办？会不会一直偏下去？

不会。协整的含义就是"偏离了会自动回来"。ECM（误差修正模型）就是描述这个"回来"的过程——回来多快、怎么回来。

2.2 ECM模型

ΔYₜ = α · ûₜ₋₁ + 短期动态项 + vₜ

ûₜ₋₁ = Yₜ₋₁ − β̂Xₜ₋₁ → 上一期偏离长期均衡的程度（误差修正项）
α → 调整速度（必须 < 0 才能纠偏）
短期动态项 → ΔXₜ 等（短期内X变化对Y的即时影响）
vₜ → 随机扰动

α 为什么必须小于零？

假设上一期Y偏高了（ûₜ₋₁ > 0，实际消费超出均衡水平）。

α < 0 → α · ûₜ₋₁ < 0 → ΔYₜ < 0 → 本期Y向下调整 → 往均衡方向回来。

反过来，上一期Y偏低了（ûₜ₋₁ < 0），α · ûₜ₋₁ > 0 → Y向上调整 → 也是往均衡方向回来。

图2：ECM的纠偏机制——α<0确保偏离后自动回归均衡

2.3 ECM的建模步骤

第一步：OLS回归 Y = β̂₀ + β̂₁X → 建立长期均衡关系
第二步：用残差 û 构建误差修正方程 → 描述短期调整机制

一句话：EG检验告诉你"有没有长期关系"，ECM告诉你"偏离了怎么回来"

⚠ 必须先确认有协整（EG检验通过），才能建ECM。没有协整就建ECM没有意义。

3. 格兰杰因果——谁能预测谁？

3.1 为什么还需要格兰杰因果？

协整告诉我们Y和X有长期关系，ECM告诉我们偏离了怎么调整。但还有一个问题没回答：

→ 到底是X引起了Y的变化，还是Y引起了X的变化？还是互相影响？

格兰杰因果检验就是回答这个问题的。

3.2 格兰杰因果的核心思想

"能预测"不等于"格兰杰原因"。格兰杰因果的定义比"能预测"更严格：

在已经用Y自身历史预测Y的基础上，再加入X的历史，如果预测变得更准了
→ X是Y的格兰杰原因

关键词：额外提高预测。不是说X能预测Y就行，而是在Y已经用了自己历史的前提下，X还能额外帮忙。

类比：你已经根据昨天的天气预报了今天的天气（Y的历史）。然后有人告诉你昨天的气压数据（X的历史）。如果加了气压后预测变准了，那气压就是天气的"格兰杰原因"。如果加了也没变准，气压就不是。

3.3 怎么检验？

构建两个方程，分别检验交叉项系数：

方程1（X能不能预测Y？）：Yₜ = Σαᵢ·Xₜ₋ᵢ + Σβᵢ·Yₜ₋ᵢ + μ₁ₜ
方程2（Y能不能预测X？）：Xₜ = Σλᵢ·Yₜ₋ᵢ + Σδᵢ·Xₜ₋ᵢ + μ₂ₜ

只看交叉项（α和λ），不看自身滞后项（β和δ）

检验X是不是Y的格兰杰原因：

H₀：α₁ = α₂ = … = αₚ = 0（X的滞后项系数全为零 → X对Y没有额外预测力）

检验方法：F检验

拒绝H₀：α ≠ 0 → X的历史值对Y有额外解释力 → X是Y的格兰杰原因
不拒绝H₀：α = 0 → 加了X也没变准 → X不是Y的格兰杰原因

3.4 四种情况

情况条件含义
X → Yα ≠ 0，λ = 0X是Y的格兰杰原因，Y不是X的
Y → Xα = 0，λ ≠ 0Y是X的格兰杰原因，X不是Y的
X ↔ Yα ≠ 0，λ ≠ 0互为格兰杰原因
无因果α = 0，λ = 0谁也不是谁的格兰杰原因

判断方法：只看交叉项（α和λ），不看自身滞后项（β和δ）

3.5 考试实例：投资额 vs 销售额

检验方向F统计量p值判断结论
投资额 → 销售额1.6810.148p=0.148 > 0.05 → 率大 → 不拒绝H₀投资额不是销售额的格兰杰原因
销售额 → 投资额9.5750.000p=0.000 < 0.05 → 率小 → 拒绝H₀销售额是投资额的格兰杰原因

白话翻译：已知投资额自身历史，再加销售额历史，对投资额的预测变准了。但反过来，加投资额历史对销售额的预测没变准。说明是销售先动，投资跟着动。

Chow检验（结构突变检验）

做法：分别对两段数据做OLS得RSS₁和RSS₂，对全样本合并回归得RSS_pooled，构造F统计量：

F = [(RSS_pooled − RSS₁ − RSS₂) / (k+1)] / [(RSS₁ + RSS₂) / (n − 2(k+1))]
H₀：两段参数相同（无结构突变）
拒绝H₀ → 存在结构突变 → 分段建模或加时间虚拟变量交叉项

Q8 什么是受约束F检验？怎么构造？
点击查看答案

用在：想检验一组约束条件是否成立（比如"某几个系数同时为零"或"两段数据系数相同"）。

F = [(RSS_R − RSS_UR) / q] / [RSS_UR / (n−k−1)]

RSS_R = 受约束模型的残差平方和（加了约束，拟合变差，RSS变大）
RSS_UR = 无约束模型的残差平方和（自由拟合，RSS较小）
q = 约束个数

H₀：约束成立
直觉：约束使RSS变大；如果增幅相对误差水平足够大 → 约束错误 → 拒绝H₀

整体F检验是特例（约束 = 所有斜率=0，此时 RSS_R = TSS）。Chow检验也是特例。

面板模型：Yᵢₜ = α₀ + β₁Xᵢₜ + uᵢ + εᵢₜ，其中 uᵢ = 个体效应。核心问题：uᵢ 和 Xᵢₜ 有没有相关？

固定效应（FE）随机效应（RE）
含义uᵢ 和 Xᵢₜ 相关（有内生性）
个体效应"固定"在某个个体上
（讲师举例：长得漂亮的uᵢ大，长得丑的uᵢ小）		uᵢ 和 Xᵢₜ 不相关（无内生性）
个体效应是随机的
（讲师举例：你大我小纯粹是概率问题）
处理uᵢ 看作截距的一部分uᵢ 看作扰动项的一部分

模型选择三步决策：

① F检验：混合 vs 固定 → H₀:混合更优（无个体效应）→ 拒绝则有个体效应
② BP检验：混合 vs 随机 → H₀:无异方差（用混合）→ 拒绝则用随机
③ Hausman检验★：固定 vs 随机 → H₀:无内生性（选RE）→ 拒绝则选FE

讲师原话：考得最多的就是Hausman检验。固定效应和随机效应是为了研究内生性的问题。

Q3 面板数据中工具变量法有何应用？工具变量的选取有什么要求？
点击查看答案

应用：面板中存在内生性时（Hausman拒绝H₀），除了用固定效应，还可以引入工具变量（IV）通过2SLS消除内生性。

讲师原话：想要解决内生性问题，是用工具变量法——引入一个工具变量，把原来存在内生性的变量给它换掉。

工具变量Z的三条要求：

  1. 相关性：Cov(Z, X) ≠ 0 → Z和内生变量X高度相关
  2. 外生性：Cov(Z, U) = 0 → Z和扰动项U不相关
  3. 排斥性：Z不直接出现在方程中（对Y的影响只通过X传递）

⚠ 只满足相关性不够！相关性+外生性缺一不可。

Q4 面板数据模型随机误差项的假设有哪些？
点击查看答案
  1. 每个截面的误差项零均值：E(εᵢₜ) = 0
  2. 每个个体对应的误差项同方差：Var(εᵢₜ) = σ²
  3. 不同截面间残差不相关：Cov(εᵢₜ, εⱼₜ) = 0, i≠j
  4. 不同截面间具有同方差

讲师原话：这个是比较死的概念性的题目，记住就OK。

对比OLS古典假定：面板假定多了"截面维度"——不仅时序残差要满足，不同个体之间也要满足。
