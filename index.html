S1. 从问题出发：OLS基础与高斯-马尔科夫定理

生活中我们想预测一件事——比如收入增加了，储蓄会增加多少？

为了回答这个问题，我们把"收入影响储蓄"写成数学关系：

总体回归方程：Y = β₀ + β₁X + U
Y = 储蓄（想预测的） |  X = 收入（原因） |  β₁ = 收入每增1元，储蓄平均增多少  |  U = 其他乱七八糟的影响（看不见）

→ 但问题来了：β₁ 到底是多少？

β₁ 是总体参数——我们永远不知道它的真实值（除非调查全国每一个人）。我们只有一份样本数据，只能用样本去估计它。估计出来的值叫 β̂₁（帽子 = "估计值"）。

→ 那怎么估计？总得有个标准吧。

标准就是：让估计出来的回归线尽可能贴近数据。怎么衡量"贴近"？看每个数据点到回归线的竖直距离（残差），让这些距离的平方和最小。

为什么用平方和？因为残差有正（点在线上方）有负（点在线下方），直接加会互相抵消。平方后全部变正，大的偏离被加倍惩罚。

图1：OLS 找到一条线，使所有红色虚线（残差）的平方和最小
最小化目标：min Σ eᵢ² = min Σ [Yᵢ − (β̂₀ + β̂₁Xᵢ)]²

怎么求解：对 β̂ 求偏导，令偏导 = 0
→ 正规方程 X'Xβ̂ = X'Y
→ 解出 β̂ = (X'X)⁻¹X'Y→ 逆矩阵来源

⚠ 2026年已考：写出 β̂ 的推导过程 + 证明无偏性。这是必须闭卷默写的公式。

→ β̂ 算出来了，但还有三个问题要回答：

  1. 残差和误差有什么区别？（先分清基本概念）
  2. 模型好不好？（解释了多少Y的变动？）→ 看 R²
  3. β̂ 准不准？（是不是在"瞄准"真值β？）→ 取决于古典假定

1.1 残差 vs 误差——先分清这两个「差」

误差 U（Error） 📖
总体角度，不可观测
Uᵢ = Yᵢ − (β₀ + β₁Xᵢ)
真实值偏离真实回归线多少
永远看不见（因为β₀, β₁未知）
残差 e（Residual）
样本角度，可以算出
eᵢ = Yᵢ − Ŷᵢ = Yᵢ − (β̂₀ + β̂₁Xᵢ)
真实值偏离估计的回归线多少
β̂已知，所以能算出来

类比：误差U是"上帝知道的真实偏离"，残差e是"我们能算出的对偏离的估计"。残差是误差的估计值。

1.2 模型好不好？—— TSS 分解与 R²

对散点图上每一个数据点 i，都可以把Y偏离均值的总距离（离差）拆成两部分：

图2：对每个点 i，总变动（TSS）= 模型解释的（ESS）+ 模型没解释的（RSS）
对所有点取平方和 → TSS = ESS + RSS

TSS = Σ(Yᵢ − Ȳ)² — Y 的总变动
ESS = Σ(Ŷᵢ − Ȳ)² — 模型解释了的部分（回归线偏离均值线多远）
RSS = Σ(Yᵢ − Ŷᵢ)² — 模型没解释的部分（数据点偏离回归线多远）

→ 那"模型好不好"就可以量化了：ESS占TSS的比例越大，模型越好。这个比例就是 R²。

R² = ESS / TSS 📖 = 1 − RSS / TSS

R² = 1：RSS = 0，所有点在回归线上 → 完美拟合
R² = 0：ESS = 0，Ŷᵢ = Ȳ，回归线退化为水平均值线 → 模型什么都没解释

R² 的缺陷和更好的指标（调整R²、AIC/BIC）放到 S4 统一讲。这里只需要知道 R² 是衡量拟合优度的基本指标。

→ 现在我们知道了β̂怎么算（OLS）、模型好不好怎么看（R²）。但最关键的问题还没答：β̂ 到底准不准？

1.3 β̂准不准？——六条古典假定

前面知道了模型好不好看R²，但这只回答了"整体拟合"。更核心的问题是：我估出来的β̂到底准不准？答案取决于古典假定是否成立。

编号假定公式直觉理解
①X 非随机X 是确定性的完全随机的东西无法预测——你拿噪声当原因，能预测出什么？
②零均值E(Uᵢ) = 0偏离有正有负，但平均为零——不系统性地偏向某一边
③同方差Var(Uᵢ) = σ²偏离的波动前后一致——不能收入低时波动小、收入高时波动大
④无自相关Cov(Uᵢ,Uⱼ)=0前一期残差不影响后一期——残差不能"手拉手"往一个方向跑
⑤正态分布Uᵢ ~ N(0,σ²)大部分偏离集中在零附近，越大的偏离越少（钟形分布）
⑥无完全共线性X'X 可逆→ 满秩条件解释变量之间不能完全线性相关——否则分不清谁在起作用

⚠ ③同方差 + ④无自相关合称"球型扰动假设"，经常一起考，缺一不可

1.3.1 全满足 → BLUE（高斯-马尔科夫定理）

满足六条假定 → β̂ 具有 BLUE 性质：

  • Best = 最优的（方差最小）
  • Linear = β̂ 是 Y 的线性函数
  • Unbiased = 无偏：E[β̂] = β（反复抽样，β̂的平均值等于真值）
  • Estimator = 估计量

一句话：在所有"Y的线性函数+无偏"的估计量中，OLS的β̂方差最小。没有比它更精确的了。

OLS 标准误的公式是 Var(β̂) = σ²·(X'X)⁻¹。这个公式是在同方差 + 无自相关的前提下推导出来的。记住这个公式——S2 讨论假设违背时，它是理解"标准误为什么会出问题"的关键。

1.3.2 无偏性证明（2026已考）

考卷给你 Yi = β₁ + β₂Xi + μi，让你写出β̂₂的公式并证明无偏。

β̂₂的公式：β̂₂ = Σ(Xi−X̄)(Yi−Ȳ) / Σ(Xi−X̄)²

Step 1：代入真实关系。Yi−Ȳ = β₂(Xi−X̄) + (μi−μ̄)，代入分子：
　Σ(Xi−X̄)(Yi−Ȳ) = β₂·Σ(Xi−X̄)² + Σ(Xi−X̄)·μi
　（利用Σ(Xi−X̄)·μ̄ = μ̄·Σ(Xi−X̄) = 0，离差和为零）

Step 2：除以分母，得到"真值+误差项"结构：
　β̂₂ = β₂ + Σ(Xi−X̄)·μi / Σ(Xi−X̄)²

Step 3：取期望，由假定 E(μi|X) = 0：
　E(β̂₂|X) = β₂  ✓ 无偏性得证 □

⚠ 关键依赖 E(μi|X)=0。若Cov(X,μ)≠0（内生性），这一步崩塌 → β̂有偏且不一致。

▶ 📖 深入理解："无偏"到底是什么意思？
一、"无偏"到底是什么意思？

好秤与坏秤。你把一个精确的1斤砝码反复放在秤上称了10次：0.98、1.02、0.99、1.01、1.00、0.97、1.03、1.00、1.01、0.99。每次结果不同，但平均恰好是1.00斤——这杆秤就叫无偏的，误差是随机的，不会系统性偏高或偏低。换一杆坏秤，每次都多0.2斤，平均1.20——这叫有偏。

OLS估计量就是那杆好秤：每次抽一批样本算出来的β̂₂都不一样，但只要古典假定成立，长期平均下来就等于真实的β₂。公式 E(β̂₂) = β₂ 只是把"长期平均等于真值"用数学符号写出来。

二、三个易混概念——一次分清
概念 数学表达 白话解释
β̂ 无偏
（估计量无偏）		 E(β̂) = β→ BLUE与估计量性质 做100次实验，每次抽一组样本算一个β̂，这100个β̂的平均值等于真实的β。β̂是随机变量（每次不一样），β是常数（真值）。
E(uᵢ|X) = 0
（残差零均值）		 E(uᵢ|X) = 0 不管X取什么值，残差平均偏离都是零——模型没有系统性遗漏。这是古典假定，用来证明β̂无偏。
E(Yₜ) = 常数
（均值不变）		 E(Yₜ) = μ, ∀t 时间序列平稳性条件之一：序列均值不随时间t漂移。跟OLS无偏性完全无关。

三者关系：E(uᵢ|X)=0 是原因（假定），E(β̂)=β 是结果（性质），E(Yₜ)=常数 是另一个话题（平稳性）。

⚠ 常见误区："无偏"≠"β̂是常数"。β̂每次算出来都不一样（随机变量），无偏只说它的期望（长期平均）等于真值。
⚠ 考试延伸：计量学里"无偏"99%指β̂无偏。还有一处：σ̂² = RSS/(n−k−1) 是σ²的无偏估计，分母用 n−k−1 就是为了 E(σ̂²) = σ²。"无偏"含义始终一样：估计量的期望 = 要估计的真实参数。
三、符号速查——谁是已知？谁是未知？
符号含义是否已知
β₂真实斜率（总体参数）❌ 永远不知道
β̂₂用样本估出来的斜率（估计量）✅ 自己算的
μᵢ随机误差项（真实世界的噪声）❌ 看不见
eᵢ残差 = Yᵢ − Ŷᵢ（μᵢ 的估计值）✅ 算出来的

关键区分：β₂ 带帽子就是估计值，μᵢ 的估计值叫残差 eᵢ。证明无偏时操作的对象是 μᵢ（真实误差），不是 eᵢ（残差）。

四、一元回归完整推导——逐步拆解（理解用）

证明思路只有两步：① 把 β̂₂ 改写成"真值 + 噪声尾巴"的形式；② 证明"噪声尾巴"的期望为零。

第①步 写出估计量公式
β̂₂ = Σ(Xᵢ − X̄)(Yᵢ − Ȳ) / Σ(Xᵢ − X̄)²
这是OLS对残差平方和求导、令其为零解出来的。这一步是纯数学，不需要任何假定。
第②步 把 Yᵢ − Ȳ 用真实模型展开
真实模型：Yᵢ = β₁ + β₂Xᵢ + μᵢ
对所有观测取平均：Ȳ = β₁ + β₂X̄ + μ̄
两式相减：Yᵢ − Ȳ = β₂(Xᵢ − X̄) + (μᵢ − μ̄)
Y 相对于均值的偏离，由两部分组成：β₂(Xᵢ − X̄) 是X变动正常带来的影响，(μᵢ − μ̄) 是随机噪声。
第③步 代入分子，展开
分子 = Σ(Xᵢ − X̄)(Yᵢ − Ȳ)
     = Σ(Xᵢ − X̄)[β₂(Xᵢ − X̄) + (μᵢ − μ̄)]
     = β₂·Σ(Xᵢ − X̄)² + Σ(Xᵢ − X̄)(μᵢ − μ̄)
第一项是β₂乘以分母（待会儿会约掉），第二项是噪声干扰。
第④步 关键恒等式消掉 μ̄
Σ(Xᵢ − X̄)(μᵢ − μ̄) = Σ(Xᵢ − X̄)·μᵢ − μ̄·Σ(Xᵢ − X̄)
恒等式：Σ(Xᵢ − X̄) = 0（离差之和永远为零——所有偏差加起来正负抵消）。
所以 μ̄ 那项整个消掉，剩下 Σ(Xᵢ − X̄)·μᵢ。
→ 这个恒等式是整个推导最关键的"工具"，考场上忘了它就推不下去。
第⑤步 拼回去，得到"真值 + 噪声尾巴"
β̂₂ = [β₂·Σ(Xᵢ − X̄)² + Σ(Xᵢ − X̄)·μᵢ] / Σ(Xᵢ − X̄)²

    = β₂ + Σ(Xᵢ − X̄)·μᵢ / Σ(Xᵢ − X̄)²
结构清晰了：第一项 β₂ 是真值（常数），第二项是被随机误差 μᵢ 干扰的"噪声尾巴"。
证明无偏，只剩一件事：证明噪声尾巴的期望等于零。
第⑥步 用假定取期望，一击必杀
E(β̂₂ | X) = β₂ + Σ(Xᵢ − X̄)·E(μᵢ|X) / Σ(Xᵢ − X̄)²
给定X后，(Xᵢ − X̄) 和分母都是常数，可以提到期望外面，只有 μᵢ 还是随机的。

由古典假定 E(μᵢ|X) = 0，噪声尾巴整个归零：
E(β̂₂|X) = β₂ + 0 = β₂
再由全期望公式：E(β̂₂) = β₂　✓
故 β̂₂ 是 β₂ 的无偏估计量。□
⚠ 无偏性的命门：整个证明只依赖 E(μᵢ|X) = 0（严格外生性）。一旦存在内生性（遗漏变量、测量误差、联立性），E(μᵢ|X)≠0，噪声尾巴的期望不为零，无偏性崩塌，且偏差不随样本量增大消失（不一致）。
五、矩阵版证明（多元回归，同样的逻辑）

多个解释变量时，逻辑完全一样，只是符号升级为矩阵：

第①步 写出估计量
β̂ = (X'X)⁻¹X'Y
第②步 代入 Y = Xβ + U，拆成"真值+噪声"
β̂ = (X'X)⁻¹X'(Xβ + U)
   = (X'X)⁻¹(X'X)β + (X'X)⁻¹X'U
   = β + (X'X)⁻¹X'U
矩阵性质：(X'X)⁻¹(X'X) = I（逆矩阵 × 原矩阵 = 单位矩阵）。
第③步 取条件期望
E[β̂|X] = β + (X'X)⁻¹X' · E[U|X]
第④步 用假定
由 E[U|X] = 0 → E[β̂|X] = β → E[β̂] = β　✓
六、考场默写版（三步速写）

考场时间紧张，写下面三步即可拿满分：

步骤①
β̂₂ = Σ(Xᵢ − X̄)(Yᵢ − Ȳ) / Σ(Xᵢ − X̄)²
步骤②
将 Yᵢ − Ȳ = β₂(Xᵢ − X̄) + (μᵢ − μ̄) 代入分子，利用 Σ(Xᵢ − X̄) = 0，化简得：

β̂₂ = β₂ + Σ(Xᵢ − X̄)·μᵢ / Σ(Xᵢ − X̄)²
步骤③
由 E(μᵢ|X) = 0 → E(β̂₂|X) = β₂ → E(β̂₂) = β₂

故 β̂₂ 是 β₂ 的无偏估计量。□

1.3.3 小样本性质与大样本性质

小样本性质就是 BLUE——不管你有多少个样本（哪怕只有 30 个），只要高斯假定成立，OLS 就是最优的。"小样本"不是说样本一定小，而是说在有限样本下就能保证。

大样本性质是一致性——当样本量 n 趋向无穷大时，β̂ 会越来越接近真实的 β，最终完全重合。一致性需要样本量足够大才能体现。

→ 这两个性质放在这里，是为了画出理想基准线：如果所有假设都满足，OLS 就是完美的。接下来的 S2 全部内容，都是在讨论假设不满足时会出什么问题、怎么检验、怎么修正。

S2. 假定被违反了：后果、检验与修正

OLS 估计完 β̂ 之后，要做一轮"体检"：逐项检查六条假定有没有被违反。发现问题 → 修正 → 修正后才能放心下结论。

主要涉及四种违背——多重共线性、异方差、自相关、内生性。在分别展开之前，先看全貌。

2.0 先看全貌——四种违背的后果对比

违反类型无偏性一致性最小方差t/F检验修正
多重共线性✓✓✗t失效，F和R²不受影响剔除变量/岭回归

→ 但三种违背对标准误的影响机制有一个关键区别，这个区别在考试中非常重要：

★ 核心区分：标准误是"真的大了"还是"算错了"？

OLS 标准误的公式是 Var(β̂) = σ²·(X'X)⁻¹，这个公式是在同方差 + 无自相关的前提下推导出来的。

多重共线性不破坏这个前提（它破坏的是满秩条件），所以公式本身仍然是对的，只不过因为 X 之间高度相关，(X'X)⁻¹ 这个矩阵的值变得很大，导致算出来的方差很大。方差大是真的大，标准误的计算没有错。

异方差和自相关直接破坏了同方差/无自相关前提。真实的方差已经不再是 σ²·(X'X)⁻¹ 了，但 OLS 不知道这一点，还在用旧公式算。算出来的标准误跟真实值可能差很远——可能偏大也可能偏小，总之是错的。

多重共线性异方差 / 自相关
标准误公式公式对，但算出来的值很大公式本身就错了
修β̂的路线剔除变量 / 岭回归WLS / 广义差分
修标准误的路线不需要（公式没错）怀特稳健标准误 / Newey-West

⚠ 这段是整个计量经济学最难理解的概念之一。如果第一遍看不懂，先记住结论（共线性 = 标准误公式对但值大，异方差/自相关 = 标准误公式本身错），后面遇到具体检验时会反复用到。

2.1 同方差假定被违反 → 异方差

什么是异方差？

同方差假定要求残差方差恒定（Var(Uᵢ) = σ²）。如果方差随X变化——比如收入越高，消费的波动越大——就叫异方差。

图3：同方差 vs 异方差的残差散点图对比

异方差检验（5种方法）

#方法做法特点
①散点图画残差²对X的图，看有没有喇叭口最直观，但不严格
②GQ检验按X排序分两段，比较两段RSS → F检验只适用递增/递减型
③White检验 ★残差²对X及其交叉项、平方项做辅助回归 → nR²~χ²最通用，不需假设形式
④Glejser检验|残差|对X回归可推断异方差函数形式
⑤BP检验残差²对X回归 → nR²~χ²比White简洁（不加交叉项）

H₀：不存在异方差（同方差） → 拒绝H₀ = 有异方差 = 需要修正

→ White检验的思路是：如果同方差成立，残差²应该跟X没关系。把残差²当Y、X及其平方项交叉项当解释变量做辅助回归。R²大 → 残差²能被X解释 → 方差跟着X变 → 异方差。

⚠ 读表时看 Obs*R-squared 那行的 Probability，不看 F-statistic 那行。

后果

OLS 仍然无偏且一致，但不再最小方差。标准误公式本身就错了（见 2.0 总览）→ t/F检验结论不可靠。

修正

  • WLS（加权最小二乘）：给方差大的观测降低权重，恢复同方差。会改变 β̂ 的值。
  • 怀特稳健标准误：不改β̂（OLS 的点估计不变），只修正标准误的计算公式 → t值变了 → 显著性结论可能变

⚠ 易混淆：怀特检验 vs 怀特稳健标准误

名字里都有"怀特"，但一个是诊断（判断有没有异方差），一个是治疗（修正标准误让检验重新可靠）。怀特检验 = 拍X光看有没有骨折；怀特稳健标准误 = 打石膏修复。

2.2 无自相关假定被违反 → 自相关

什么是自相关？

无自相关假定要求不同观测的误差互不相关。如果前一期残差影响后一期——残差"手拉手"往一个方向跑——就叫自相关。时间序列数据尤其常见。

自相关检验（2种）

① DW检验（Durbin-Watson）— 只能检验一阶自相关

DW ≈ 2(1 − ρ̂)，ρ̂ = 相邻残差的相关系数
ρ̂=1 → DW=0（完全正自相关）| ρ̂=0 → DW=2（无自相关✓）| ρ̂=−1 → DW=4（完全负自相关）
图4：DW检验的区间判断

DW 的四个限制条件：①模型必须有截距项且不含被解释变量的滞后项；②解释变量必须是非随机的；③残差是正态分布且只能检验一阶自回归；④样本容量要足够大。不满足这些条件就不能用 DW，改用 LM 检验。

② BG-LM检验 ★ 更通用— 可以检验高阶自相关

做法：残差对自身滞后项做自回归 → nR² ~ χ²

H₀：不存在自相关 → 读表看 Obs*R-squared 行：p < 5% → 有自相关

⚠ 考试陷阱：DW 约等于 4 是负自相关，不是正自相关！

后果

和异方差一样：OLS仍无偏一致，但标准误公式本身错了（见 2.0 总览），t/F不可靠。

修正

  • 广义差分：对Y和X同时做 (Yₜ − ρ̂Yₜ₋₁) 变换，消除自相关后重新OLS
  • Cochrane-Orcutt迭代法：先估 ρ̂ ≈ 1 − DW/2，再做广义差分
  • Newey-West 稳健标准误：类似怀特稳健标准误的思路，不改 β̂，只修正标准误。异方差用怀特，自相关用 Newey-West。

⚠ ρ̂ ≈ 1 − DW/2（不是DW本身！）考过：DW=0.52 → ρ̂ = 1−0.52/2 = 0.74，不是0.52

→ 关联：金融计量学恰恰要利用自相关来建模预测（→ 第二条线的 AR/MA/ARMA），而不只是"修正"它。自相关是连接计量经济学和金融计量学的桥梁。

2.3 无完全共线性假定被违反 → 多重共线性

什么是多重共线性？

解释变量之间高度线性相关。比如"身高"和"臂展"同时放进回归——两者几乎成正比，模型分不清谁在起作用。

判断方法（4种，不是假设检验的形式，没有H₀）

#方法判断标准
①简单相关系数|Rᵢⱼ| ≈ 1 → 两个变量高度相关（只能看两两关系）
②VIF（方差膨胀因子）VIF = 1/(1−R²ⱼ) > 10 → 严重。R²ⱼ 是把 Xⱼ 对其余X回归的R²
③F大R²大但t小整体显著但各自不显著 → 典型共线性症状
④条件指数CICI > 20 → 有共线性

R² 和 R²ⱼ 的区别：R² 的被解释变量是 Y，衡量模型对 Y 拟合好不好；R²ⱼ 的被解释变量是 Xⱼ，衡量 Xⱼ 跟其他 X 有多相关。两者被解释变量都不一样，不要搞混。

后果（推导链——简答必背）

完全共线性：X'X不可逆 → 无法求出β̂（直接炸了）

不完全共线性推导链：
解释变量间相关性高 → R²ⱼ变大 → VIF↑ → Var(β̂)↑
→ ① 不再具有最小方差性（BLUE被打破）
→ ② t = β̂/se(β̂) ↓（分母se变大）→ t检验失效
→ ③ 置信区间变宽 → 预测精度下降

★ 为什么共线性只影响 t 不影响 F？

类比：三个人合伙搬一块大石头。石头搬动了（F检验通过，说明三个人合在一起有用），但没办法分清每个人出了多少力（t 检验失灵），因为他们的动作纠缠在一起了。

严格来说：t 检验考察的是单个变量的独立贡献，共线性让变量之间的贡献分不清，所以 t 失灵。但 F 检验考察的是所有变量联合在一起的解释力，共线性只是变量间信息重叠，总的解释力并没有减少，所以 F 不受影响。

修正

剔除不重要变量 / 逐步回归（AIC/BIC选变量）/ 岭回归（牺牲一点无偏性换取方差大幅减小）

2.4 外生性假定被违反 → 内生性（最严重！）

什么是内生性？

解释变量X和扰动项U相关：Cov(X, U) ≠ 0。

⚠ 易混淆：内生性 vs 自相关

自相关是残差之间的关系（Cov(uᵢ,uⱼ)≠0），E(u|X)=0 仍然成立，所以 OLS 仍然无偏且一致，只是标准误算错了。

内生性是 X 与残差的关系（Cov(X,u)≠0），E(u|X)≠0，无偏性证明的关键一步直接崩塌，OLS 有偏且不一致。

一句话：自相关是"精度"问题（估计值对但不够精确），内生性是"方向"问题（估计值本身就是歪的，样本再大也纠正不了）。这是所有假设违背中最严重的一种。

三种来源

  1. 遗漏变量（最常见）：重要变量被遗漏进入U，且与X相关。例：研究教育对收入的影响，但"能力"无法观测，被遗漏进入U，而能力和教育年限相关
  2. 测量误差：X的观测值含误差，误差进入U且与X相关
  3. 联立性/反向因果：X影响Y的同时Y也影响X（如价格与需求互相决定）

检验

Hausman检验：如果没有内生性，OLS和IV结果应该差不多。如果差很大 → 有内生性。

H₀：不存在内生性

拒绝H₀：有内生性 → OLS有偏且不一致！
不拒绝H₀：外生OK ✓，用OLS就行

后果

OLS 有偏且不一致——唯一破坏无偏性和一致性的违反！

修正：工具变量法（IV）/ 两阶段最小二乘（2SLS）

核心思路：X"脏了"（和U相关），找一个"干净的替身"Z。Z必须同时满足：

  1. 相关性：Cov(Z, X) ≠ 0 → Z和X高度相关
  2. 外生性：Cov(Z, U) = 0 → Z和U不相关
2SLS 两步操作：
第一步：用Z回归X → 得到拟合值X̂（X̂只保留了X中被Z解释的"干净"部分）
第二步：用X̂替代原来的X，对Y做OLS → 得到一致的β̂

代价：IV估计的方差比OLS大——用效率换一致性。

⚠ 只满足相关性不够！相关性+外生性缺一不可。

2.5 补充：随机误差项U包含哪些因素？（名词解释题）

  1. 模型中省略的解释变量的影响
  2. 研究变量的测量误差
  3. 随机因素（X、Y本身的随机波动）
  4. 经济活动中的某些不确定性

理解：U是"垃圾桶"，模型没能解释的东西全在里面。

S3. 假设检验：所有检验共用的底层逻辑

S2 提到了很多检验——White、GQ、DW、LM、Hausman…它们看起来各不相同，但底层逻辑只有一套。把这套逻辑抽出来单独理解，就不会把不同的检验当成孤立的知识点去死记硬背。

3.1 通用四步流程（万能模板）

① 列假设：写出 H₀ 和 H₁
② 构造统计量：代入数据算出 t / F / χ² 的数值
③ 查临界值：根据α和自由度，查分布表得到门槛值
④ 比较判断：|统计量| vs 临界值 → 套口诀

口诀：数小率大接H₀，数大率小拒H₀
"数" = |统计量| vs 临界值 → 统计量 < 临界值 = "数小" → 不拒绝
"率" = p值 vs α → p > α = "率大" → 不拒绝

类比：H₀ 相当于无罪推定，统计量是证据的份量，临界值是定罪标准。证据够（统计量大）就定罪（拒绝 H₀），证据不够（统计量小）就无罪释放（接受 H₀）。

3.3 自由度从哪来

k = 回归模型中解释变量的个数（不含截距 β₀）。如果模型是 Y = β₀ + β₁X₁ + β₂X₂ + β₃X₃ + u，那 k=3。

t 检验：自由度 = n − k − 1
F 检验（联合显著性）：F(k, n−k−1)
χ² 检验（White / LM）：自由度 = 辅助回归中解释变量的个数

⚠ 自由度算错是计算题常见坑：F(k, n−k−1) 中的 k 不含截距，n−k−1 中的 1 是截距的自由度。

S4. 模型评判与预测

检验都通过了（或已修正），最后一步：模型整体好不好？能不能用来预测？

4.1 拟合优度指标

指标公式方向注意
R²ESS / TSS = 1 − RSS/TSS越大越好增加变量R²必然不减，无法判断变量有没有用
调整R²（R̄²）1 − (1−R²)·(n−1)/(n−k−1)越大越好对变量数量k有惩罚：加入无用变量R̄²会下降
AICln(σ̂²) + 2k/n越小越好模型选择指标
BIC（SIC）ln(σ̂²) + k·ln(n)/n越小越好惩罚比AIC重，AIC与BIC不一致时优先BIC

为什么加变量 R² 必然不减？

这是数学上的必然。OLS的目标是 min RSS，多一个变量等于多一个可调节的"旋钮"，OLS至少可以把这个变量的系数设为零（等于没加），所以 RSS 只可能不变或更小，R² = 1 − RSS/TSS 就只能不变或更大。调整R²通过自由度惩罚修正了这个问题：新变量如果没用，RSS只微降，但惩罚加重，调整R²反而下降。

AIC vs BIC 怎么选？

两者都是"拟合优度 − 复杂度惩罚"的权衡。BIC的惩罚项是 k·ln(n)，当样本量n较大时（n>7 即 ln(n)>2），惩罚比AIC的 2k 更重，更倾向简约模型。两者不一致时，优先BIC。

模型选择指标优先级：BIC > AIC > 调整 R²。

4.2 预测：点预测与区间预测

点预测
给定 Xf，直接代入方程：
Ŷf = β̂₀ + β̂₁Xf
得到Y的单个预测值
区间预测
给定置信水平(1−α)，求Y落在的区间：
Ŷf ± tα/2 · se(预测误差)
区间越宽 = 预测精度越低

预测区间总是比估计区间宽，因为预测不仅包含参数估计的不确定性，还包含残差本身的随机波动。当假设被违反（如存在异方差、自相关），方差估计偏大，置信区间变宽，区间预测失效——与 S2"后果三条"完全对应。

从第一条线到第二条线：为什么需要时间序列？

在第一条线里，我们学了OLS回归的基础。OLS有一条假定叫"无自相关"——要求残差前后期互不影响。如果违反了，我们把它当作"问题"去修正。

→ 但金融数据恰恰就是自相关的！

股价今天涨了，明天大概率还涨；今天跌了，明天大概率还跌。这种"惯性"就是自相关。如果我们只是把自相关当"问题"去消除，等于扔掉了最有价值的信息。

时间序列分析的核心思想：不是消除自相关，而是利用自相关来建模预测。把序列拆成两部分：

确定部分（可预测）
序列有自相关 + 平稳
→ 用AR/MA/ARMA建模
→ 历史信息可以预测未来
随机部分（不可预测）
残差无自相关 + 平稳
→ 希望是白噪声
→ 说明确定部分已提取干净

类比：序列像一首歌，"确定部分"是旋律（有规律可循），"随机部分"是底噪（完全随机）。建模就是把旋律识别出来，剩下的纯噪音说明旋律已经提取完了。

→ 整个时间序列建模就是一个固定流程，四步走完：

下面逐步展开。每一步都是：为什么要做这一步 → 怎么做 → 结果怎么判断 → 不通过怎么办。

第一步：序列平不平稳？ 📖 白噪声三条件 📖 平稳性三层次 📖 对数差分≈收益率

1.1 为什么第一步就要看平稳性？

平稳的意思是：序列的统计特征（均值、方差、自协方差）不随时间变化。这意味着过去的规律未来还会继续——这是预测的前提。

如果序列不平稳（比如股价一直在涨），过去的规律不一定适用于未来。更严重的问题是：两个不平稳序列直接回归会产生伪回归——R²虚高但无真实关系。

→ 所以拿到数据第一件事：检查平不平稳。

1.2 平稳性的三个条件（宽平稳）

① 均值恒定：E(Yₜ) = μ，不随时间变（没有趋势）
② 方差恒定：Var(Yₜ) = σ²，不随时间变（波动幅度稳定）
③ 自协方差只取决于滞后阶数k，不取决于时点t：Cov(Yₜ, Yₜ₋ₖ) = γₖ

直觉：平稳序列像围绕一条水平线上下波动，波动幅度前后差不多。非平稳序列像一直往上走或波动越来越大。

③ADF检验 ★统计假设检验最严格，考试必用

1.4 ADF检验（核心）📖 φ的作用(AR1) 📖 滞后算子L 📖 根与系数倒数 📖 ADF检验设定

先从最简单的情况说起：AR(1)

AR(1)模型：Yₜ = φ·Yₜ₋₁ + εₜ。展开后，每期历史冲击 εₖ 传到今天被乘以 φᵏ 作为权重：

|φ| < 1
φᵏ 随k增大快速衰减
历史冲击影响消失
→ 平稳
φ = 1
φᵏ 永远等于1
每期冲击100%保留，永不衰减
→ Var(Yₜ)=tσ² 随时间增大
→ 非平稳（随机游走）
|φ| > 1
φᵏ 越来越大
冲击被放大 → 爆炸
→ 更不平稳

φ=1就是"单位根"——平稳与不平稳的分界线。过去的冲击永不衰减，像喝醉酒的人越走越偏。

高阶AR(p)：用特征方程

AR(1)只有一个系数φ，直接看就行。但AR(p)有多个系数（φ₁, φ₂, …, φₚ），怎么判断？

→ 引入滞后算子L（或B），把所有滞后项整理成一个多项式方程：

滞后算子：L·Yₜ = Yₜ₋₁，L²·Yₜ = Yₜ₋₂，…

AR(p)整理后：(1 − φ₁L − φ₂L² − … − φₚLᵖ) = 0

平稳条件：这个多项式方程的所有根的模 > 1（都在单位圆外）
有根的模 = 1 → 有单位根 → 非平稳
有根的模 < 1 → 爆炸 → 非平稳

⚠ 根和系数是倒数关系：根 L = 1/φ。|φ|<1（衰减→平稳）等价于 |L|>1（根在圆外）。"根在圆外"和"系数小于1"说的是同一件事。

本身就平稳 → I(0)

实际中常用对数差分：先取对数 ln(Pₜ)，再差分 ln(Pₜ) − ln(Pₜ₋₁)
对数差分 ≈ 收益率（当变化率很小时，ln(1+x) ≈ x）
例：股价序列（非平稳）→ 取对数 → 差分 → 收益率序列（平稳）→ 可以建模了

第二步：序列是什么类型的自相关？

2.1 为什么要识别类型？

平稳序列有自相关性（否则就是白噪声，不值得建模）。但自相关有不同形式——"用过去的自己预测自己"和"用过去的误差预测自己"是两种不同的机制，对应不同的模型。选错了模型，预测就不准。

→ 怎么区分？用ACF和PACF两个工具。

2.2 ACF和PACF是什么？

ACF（自相关函数）= 总相关 📖 📖
把序列往后错开k期，看错开前后两段有多像
包含直接影响 + 间接传递（全部路径的影响都算在里面）
k越大（错开越远）→ 相关性越弱 → "近大远小"
PACF（偏自相关函数）= 直接相关
只看 Yₜ 和 Yₜ₋ₖ 两个点之间的净关系
把中间时期的传递效应扣掉了
类比求偏导：其他变量按住不动，只看这一个

ACF = 直接影响 + 间接传递 = 总效果。PACF = 只留直接影响。所以PACF一定不会比ACF大。

公式层面的区别：ACF的分母都除以γ₀（Yₜ自己的方差），间隔k的间接影响混在里面；PACF的分母分别除以Yₜ和Yₜ₋ₖ各自的标准差，点到点归一化，只反映直接关系。

模型公式核心思想直觉
AR(p)Yₜ = c + φ₁Yₜ₋₁ + … + φₚYₜ₋ₚ + εₜ用过去的自己预测现在今天股价受昨天、前天股价影响——动态惯性

ARMA(p,q)两者结合既受历史自身影响，又受历史冲击影响两种机制兼有
ARIMA(p,d,q)先差分d次变平稳，再建ARMA不平稳序列的处理方式股价本身非平稳→取收益率（差分）→平稳→建模

  • AR模型 → 需要看特征根（是否都在单位圆外）
  • ARMA → 检验其中AR部分

  • 方法一：看ACF/PACF图，几阶后进入置信区间 → 就是几阶
  • 方法二：信息准则，在候选模型中选AIC/BIC最小的
AIC = ln(σ̂²) + 2k/n
BIC = ln(σ̂²) + k·ln(n)/n

BIC惩罚更重（n>7时 ln(n)>2），倾向更简洁的模型
AIC与BIC不一致时，优先参考BIC（避免过拟合）

→ 模型选好了、阶数定好了、参数估计完了。接下来要回答：模型拟合得好不好？

第三步：残差是白噪声吗？（模型拟合好不好？）

3.1 为什么要检验残差？

建完模型后提取残差 ε̂ₜ = Yₜ − Ŷₜ。如果模型把"确定部分"（自相关结构）都提取干净了，剩下的残差应该是纯随机的——也就是白噪声。

如果残差不是白噪声（还有自相关），说明模型没提取干净，需要调整阶数重来。

3.2 白噪声三条件

① 均值恒定：E(εₜ) = 0
② 方差恒定：Var(εₜ) = σ²（同方差）
③ 完全无自相关：Cov(εₜ, εₛ) = 0, t≠s

白噪声 = 宽平稳 + 无自相关。比宽平稳更严格（宽平稳允许有自相关，白噪声不允许）。

严格程度排序：严格平稳 > 白噪声 > 宽平稳

3.3 怎么检验？

① ACF图观察法：画残差的ACF图，所有ACF都在置信区间内 → 白噪声

② Box-Pierce检验（Q_BP）

统计量：Q_BP = n · Σρ̂ₖ² ~ χ²(m)

③ Ljung-Box检验（Q_LB）★ BP的改进版，更准

Q_LB = n(n+2) · Σ[ρ̂ₖ²/(n−k)] ~ χ²(m)

H₀：残差是白噪声

不拒绝H₀（p > 5%）→ 残差是白噪声 → 模型OK ✓
拒绝H₀（p < 5%）→ 残差不是白噪声 → 调整模型阶数重来

⚠ 白噪声检验的H₀方向特殊：H₀是"好消息"（是白噪声），不拒绝H₀才是好结果！和大多数检验相反。

3.4 白噪声的三大应用场景（考试爱出辨析题）

对什么做白噪声检验问的问题是白噪声不是白噪声
① 对平稳序列做序列能预测吗？完全随机，不值得建模有自相关，可以建模 ✓
② 对模型残差做模型拟合好吗？信息提取干净 ✓还有信息没提取，调整模型
③ 对ε²做波动率恒定吗？方差恒定，无ARCH效应 ✓有ARCH效应，需建GARCH

同一个工具用在不同对象上，问的问题不同，"好消息"的方向也不同。第③种就是下一步要做的。

→ 残差是白噪声了（均值方程OK），但还没完——金融数据有个特殊现象：波动聚集。

第四步：波动率恒定吗？（需不需要给方差建模？）

4.1 什么是波动聚集？

金融数据有个显著特征：大波动之后往往跟着大波动，小波动之后跟着小波动。虽然残差的均值为零、无自相关（看起来像白噪声），但方差不恒定——波动率在变化。

图3：金融序列的波动聚集——残差均值为零但方差在变

这意味着方差本身也是可以预测的——用过去的波动预测未来的波动。ARCH/GARCH就是给方差建的模型。

→ 先检验有没有这种现象（ARCH效应），有才建模。

4.2 ARCH-LM检验

检验什么：残差的平方 ε̂²ₜ 有没有自相关。如果ε̂²ₜ有自相关 = 波动率可预测 = 有ARCH效应。

做法：ε̂²ₜ 对自身滞后项做自回归 → nR² ~ χ²(q)

H₀：不存在ARCH效应（所有αᵢ = 0）

不拒绝H₀ → 无ARCH效应 → 方差恒定 → 建模完成 ✓
拒绝H₀ → 有ARCH效应 → 波动率可预测 → 需要建GARCH模型

也可以对ε̂²ₜ做LB白噪声检验：ε̂²ₜ不是白噪声 → 有ARCH效应。

4.3 ARCH模型

ARCH的思路：把时变的方差σ²ₜ也当作一个可以被预测的变量来建模——用过去的残差平方（冲击大小）预测未来的方差。

ARCH(q)方差方程：
σ²ₜ = α₀ + α₁ε²ₜ₋₁ + α₂ε²ₜ₋₂ + … + αqε²ₜ₋q

含义：今天的波动率 = 一个基准水平(α₀) + 过去q天冲击大小的加权和
昨天冲击大 → 今天方差大 → 今天大概率还是大波动

→ 但ARCH有个问题：要精确刻画波动率，可能需要很多滞后项（q很大）→ 参数太多。怎么简化？

4.4 GARCH模型（ARCH的升级版）

GARCH在ARCH基础上加了一项：方差自身的滞后值。这样只需要少量参数就能捕捉长期的波动持续性。

GARCH(p,q)方差方程：
σ²ₜ = ω + α₁ε²ₜ₋₁ + … + αqε²ₜ₋q + β₁σ²ₜ₋₁ + … + βpσ²ₜ₋p

ω = 长期均值水平
αᵢε²ₜ₋ᵢ = 过去冲击的影响（ARCH项 ≈ MA的思路）
βⱼσ²ₜ₋ⱼ = 过去波动率的影响（GARCH项 ≈ AR的思路）

本质：ARMA用在方差上——就像均值方程用ARMA，方差方程也用类似结构

ARCH vs GARCH 怎么区分？

ARCH(q)GARCH(p,q)
方差方程只有 ε²项（过去冲击）同时有 ε²项 + σ²项（过去冲击 + 过去方差）
读参数q = ε²滞后项个数p = σ²滞后项个数，q = ε²滞后项个数
判断方法方差方程里没有σ²ₜ₋ⱼ → ARCH方差方程里有σ²ₜ₋ⱼ → GARCH

记忆：GARCH(p,q) 中 p对应σ²（方差自回归），q对应ε²（冲击项）。看求和符号上界直接读出p和q。

GARCH(1,1)参数约束（考试必写）

σ²ₜ = ω + αε²ₜ₋₁ + βσ²ₜ₋₁

参数约束：
ω > 0（基准方差为正）
α ≥ 0，β ≥ 0（权重非负）
α + β < 1（弱平稳条件——严格小于！不是≤）

α + β 越接近1 → 波动持续性越强（冲击衰减越慢）

⚠ α + β < 1 是严格不等号！判断题考过"α+β≤1"——错！必须严格小于。

S5 从一维到多维——多序列关系分析

到目前为止处理的都是单个序列。但现实中经常面对多个序列：股价和利率、GDP和通胀……多个序列之间有没有关系？能不能互相帮助预测？这一节把S2-S4的一维思路拓展到多维。

⚠ 为什么不能直接对非平稳序列做回归？——伪回归

两个各自有上升趋势的非平稳序列直接做回归，会得到很高的R²和显著的t，但它们之间可能毫无真实关系——只是碰巧都在涨。这叫伪回归（Spurious Regression）。

判断标志：R² > DW → 高度怀疑伪回归

先检验平稳性（或协整关系），才能做有意义的回归。这是做ADF检验的核心动机。

5.1 协整（Cointegration）

问题起点：两个序列X₁和X₂各自都是不平稳的I(1)序列，但也许它们之间存在某种长期关系？

协整定义：若两个I(1)序列的某个线性组合
Z = aX₁ + bX₂ 是平稳的I(0)
则称X₁和X₂之间存在协整关系

经济含义：长期均衡。短期内两个变量可能各自偏离，但存在一根"隐形的绳子"，偏离太远就会被拉回来。
例：现货价格与期货价格——短期可能有偏差，长期必然趋于一致。

术语对比：单整 vs 协整

概念对象含义
单整 I(d)单个序列自己差分d次后平稳
协整多个序列各自不平稳，但线性组合平稳

5.2 EG两步法（检验一阶协整）

第一步：对Y和X做OLS回归 → 得到残差 û = Y − β̂X

第二步：对残差 û 做ADF单位根检验
H₀：无协整（残差不平稳）→ 拒绝H₀ = 有协整 = 好消息

⚠ 此处ADF临界值与普通ADF不同（残差是估计量，标准更严格）
⚠ EG只适用两个变量；多个变量用Johansen检验

逻辑：若Y和X有长期均衡关系，则Y偏离β̂X的部分（残差）应在均衡值附近波动，不会越跑越远——即平稳的。

5.3 误差修正模型 ECM

协整回答了"有没有长期均衡关系"。ECM进一步问：短期偏离了长期均衡，怎么修正回去？

ΔYₜ = α · ûₜ₋₁ + 短期动态项 + vₜ

ûₜ₋₁ = Yₜ₋₁ − β̂Xₜ₋₁（上一期偏离长期均衡的程度）

关键：α < 0（纠偏机制）
上期Y偏高（û > 0）→ α < 0 使ΔY < 0 → 本期Y向下调整回均衡
|α| 越大 → 纠偏越快；|α| 越接近0 → 纠偏越慢

EG与ECM的关系

EG检验告诉你"有没有长期关系"，ECM告诉你"偏离了怎么回来"。必须先通过EG检验确认协整，才能建ECM。

5.4 VAR 模型（向量自回归）

协整和ECM处理两个序列的均衡关系。VAR更一般：让多个序列互相影响、互相预测。

VAR(p)：Yₜ = c + A₁·Yₜ₋₁ + A₂·Yₜ₋₂ + … + Aₚ·Yₜ₋ₚ + uₜ

Yₜ = 包含所有变量的向量（如[股价, 利率, 汇率]ᵀ）
A₁…Aₚ = 系数矩阵

5.5 Granger 因果检验

VAR告诉我们"多个序列互相影响"，但没回答方向。格兰杰因果检验回答：到底是X影响Y，还是Y影响X？

核心思想：在已知Y自身历史的基础上，加入X的历史信息后预测效果显著提高
→ X是Y的Granger原因

检验框架：
方程1：Yₜ = c + Σαⱼ·Yₜ₋ⱼ + Σγⱼ·Xₜ₋ⱼ + uₜ
方程2：Xₜ = c + Σδⱼ·Xₜ₋ⱼ + Σλⱼ·Yₜ₋ⱼ + vₜ

H₀：γ₁ = … = γₚ = 0（X的滞后项对Y无额外预测力）
F检验，拒绝H₀ → X是Y的Granger原因
情况γ联合检验λ联合检验结论
① X → Y拒绝（γ≠0）不拒绝（λ=0）X单向影响Y
② X ← Y不拒绝（γ=0）拒绝（λ≠0）Y单向影响X
③ X ↔ Y拒绝（γ≠0）拒绝（λ≠0）互为因果
④ X ⊥ Y不拒绝（γ=0）不拒绝（λ=0）完全无关

⚠ Granger因果是统计预测关系，不是真实因果关系！考试判断题高频考点。结论依赖模型设定，换滞后阶数可能翻转。

面板数据：个体 × 时间的联合分析

面板数据（panel data）是时间序列数据的"横向扩展"：时间序列针对一个个体在多个时点上的观测，面板数据是多个个体在多个时点上的观测。符号约定：i = 个体编号（1…N），t = 时间编号（1…T），共 N×T 个观测值。

为什么从时间序列走到面板数据？

P1-P2 学 OLS 与假定；P2 学假设违反与修正（异方差/自相关/多重共线性）；P3 学时间序列（单位根、AR/MA、ARCH/GARCH）——这些都针对单个个体。当个体从1变成N，新问题出现：个体之间存在异质性，若不处理，就是遗漏变量偏误。这是本章的核心挑战。

→ 面板数据建模的核心是一棵决策树，两步检验选定模型：

下面各节逐步展开。每一步都是：为什么要做 → 怎么做 → 结果怎么判断 → 不通过怎么办。

§1 面板数据的两种效应

视频用"全班同学发传单"来说明面板数据天然包含的两种差异来源：

时间效应（time effect）

同一个人在不同月份业绩不同。
10月天暖路人愿停留 → 业绩好；
12月天冷行色匆匆 → 业绩差。

"因为时间不同导致的差异"——P3时间序列已涉及
个体效应（individual effect）

同一月份，不同人业绩也不同。
A长得漂亮口才好 → 路人愿接传单；
C性格内向社恐 → 业绩自然差。

"因为个体不同导致的差异"——本章新增维度，也是核心挑战

💡 这两个例子用的是同一个场景，同时展示了两种效应。面板数据天然同时包含这两种差异来源，建模时必须同时考虑。

§2 核心问题：个体异质性

个体异质性（individual heterogeneity）就是说不同个体之间存在本质上的不同，而且往往不可观测、无法量化。"长得漂亮""性格外向"这些因素确实影响业绩，但没法把它们变成X放进回归方程。

⚠ 不处理个体异质性会怎样？

把所有人的数据混在一起做普通OLS，A因为漂亮多出来的业绩、C因为社恐少掉的业绩，都被塞进了残差里。残差不再是纯随机噪声——里面混进了"A天生就比C强"这种系统性因素。

这就违反了 OLS 严格外生性假定 E[u|X]=0，本质是 P2讲过的遗漏变量偏误——遗漏了和X相关的变量（个体特征），它躲在残差里，导致内生性，OLS估计有偏且不一致。

§3 复合扰动项的分解

面板数据的解决方案：把扰动项拆成两部分，专门用一个项来捕捉个体异质性。

面板数据总体方程：
Yit = β₀ + β₁Xit + vit

复合扰动项：vit = ui + εit

ui：个体效应——只带下标i，不带t；不随时间变化（A今天漂亮，明天、后天还是漂亮）；使每个个体拥有不同的"起跑线"（截距不同）

εit：随机扰动——同时带i和t；纯随机波动，零均值同方差无自相关

这个分解是理解三种模型的钥匙：围绕"ui存不存在"以及"ui跟X有没有关系"，衍生出了三种不同的建模策略。

§4 ★ 三大模型总览

面板数据建模的核心选择归根到底就是一个问题：怎么看待个体效应 ui？

比较维度混合回归（Pooled OLS）固定效应（FE）随机效应（RE）

4.1 混合回归模型（Pooled OLS）

最简单的处理：假设所有个体没有差异，直接把面板数据当一个大截面做普通OLS。所有个体共享同一个截距β₀和同一个斜率β₁。

Yit = β₀ + β₁Xit + εit

好处是简单。坏处：如果个体效应确实存在（A就是比C强），混合回归把这种差异塞进残差，导致遗漏变量偏误。只在"个体之间确实没有系统性差异"时才合适，需通过F/BP检验确认。

4.2 ★★ 固定效应模型（Fixed Effects, FE）

承认个体效应存在，且认为ui与Xit相关——存在内生性。

例：A的业绩起点高，是因为A长得漂亮。而"漂亮"可能和X中的"互动时长"相关——漂亮的人更容易让路人停下来，互动时长更长。这样ui（漂亮）和Xit（互动时长）就产生了关联，即内生性。

处理方法：把ui视为截距的一部分。由于ui不随时间变化，通过"去均值"（within transformation）将其消除：

去均值变换：
Yit − Ȳi = β₁(Xit − X̄i) + (εit − ε̄i)

对每个个体的每个变量减去该个体的时间均值，ui就被差掉了。去均值后再做OLS，估计量一致。
个体固定效应
只控制个体差异（A和C不一样），不控制时间差异
时间固定效应
只控制时间差异（10月和12月不一样），不控制个体差异
双向固定效应
同时控制个体和时间差异。实际论文最常用

⚠ 局限：去均值消除ui的同时，也消除了所有不随时间变化的变量（性别、行业等）。若想估计这些时不变变量的系数，FE做不到。

4.3 随机效应模型（Random Effects, RE）

承认个体效应存在，但认为ui与所有Xit不相关——个体差异是纯随机的，跟任何可观测因素无关。

例：每个人业绩确实不同，但这种不同不是因为漂亮或社恐这些与X相关的因素，而纯粹是"运气"——有人运气好碰到的路人更友善，这种差异是随机抽到的，跟X无关。

Yit = β₀ + β₁Xit + (ui + εit)

ui ~ iid(0, σ²u)，与Xit不相关
估计方法：GLS（广义最小二乘）
优点
可估计不随时间变化的变量（性别/行业）的系数；效率更高
风险
若ui实际上与X相关（有内生性），RE估计有偏不一致。必须用Hausman检验确认假设是否成立

§5 ★★★ 模型选择：两步检验决策树（高频考点）

第一步：F检验——有没有个体效应？

H₀：所有个体截距相同（混合模型正确）
H₁：至少部分个体截距不同（FE或RE更合适）

拒绝H₀ → 个体差异存在 → 进入第二步。不拒绝H₀ → 混合OLS，到此结束。

补充：BP检验（Breusch-Pagan）从异方差角度做同样判断：H₀: σ²_u=0（无个体效应）vs H₁: σ²_u>0（有个体效应，选RE方向）

第二步：Hausman检验——固定 vs 随机？（最重要）

H₀：ui与X无关（无内生性）→ FE和RE都一致，但RE更有效 → 选随机效应
H₁：ui与X相关（有内生性）→ 只有FE一致，RE有偏 → 选固定效应

原理：若无内生性，β̂FE和β̂RE应该很接近；若两者差距大，说明RE产生了偏误，只有FE可靠。

H = (β̂FE − β̂RE)' [Var(β̂FE) − Var(β̂RE)]⁻¹ (β̂FE − β̂RE) ~ χ²(k)

口诀：Hausman显著（拒绝H₀）→ 固定效应；Hausman不显著（不拒绝H₀）→ 随机效应

检验名称H₀H₁拒绝H₀结论用途
F检验所有截距相同（混合）截距不同（FE）选FE或RE判断个体效应是否存在
BP检验σ²_u=0（混合）σ²_u>0（随机）选RE方向从异方差角度判断个体效应
Hausman检验ui与X无关（RE）ui与X相关（FE）选FE判断内生性，FE vs RE

§6 实证研究套路

面板数据实证有一套固定流程，与P3时间序列高度平行：

第一步：描述性统计 — 均值、标准差、最大最小值，了解数据基本分布

第二步：平稳性检验 — 对每个变量Xit做平稳性检验（LLC检验、IPS检验等），核心思想与P3的ADF单位根检验完全一致：检验特征方程是否有单位根，H₀:存在单位根（非平稳），拒绝H₀才说明平稳

第三步：模型识别 — F/BP检验（有无个体效应）→ Hausman检验（FE vs RE），即§5决策树

第四步：参数估计与检验 — 确定模型后做估计，对系数做t检验/F检验，确定固定效应类型（个体/时间/双向）

第五步：稳健性检验 — 更换变量度量方式、更换估计方法、更换样本区间，验证结论是否稳健（学术论文必做，考试了解即可）

S1 OLS基础

Q1 简述OLS的基本假设及其含义 简答 ★★★ 高频
展开答案

答案

OLS的基本假设（古典假定）有六条：①解释变量X是非随机的（在重复抽样中固定不变）；②随机误差项均值为零，E(Uᵢ)=0（偏离有正有负，平均为零）；③同方差，Var(Uᵢ)=σ²（每个观测的残差波动幅度一致）；④无自相关，Cov(Uᵢ,Uⱼ)=0, i≠j（不同观测的残差互不影响）；⑤随机误差项服从正态分布，Uᵢ~N(0,σ²)；⑥解释变量之间无完全共线性（X'X可逆）。

以上六条全部满足时，由高斯-马尔科夫定理可知OLS估计量是BLUE（最佳线性无偏估计量），即在所有线性无偏估计量中方差最小。加上假定⑤正态性后，还可以做精确的t检验和F检验。

知识点总结

编号假定公式白话
①X非随机X确定性解释变量固定
②零均值E(Uᵢ)=0偏离平均为零
③同方差Var(Uᵢ)=σ²波动幅度一致

Q2 设回归模型为 Yi = β₁ + β₂Xi + μi，(1)写出β̂₂的计算公式；(2)证明β̂₂是β₂的无偏估计量。 推导 ★★★★★ 2026真题方向
展开推导步骤

答案

(1) β̂₂的计算公式

OLS的目标是最小化残差平方和 Σ[Yi − (β̂₁ + β̂₂Xi)]²。分别对β̂₁和β̂₂求偏导令其为零，解正规方程组，得到：

β̂₂ = Σ(Xi − X̄)(Yi − Ȳ) / Σ(Xi − X̄)²

直觉：分子是X和Y的共变动，分母是X自身的变动幅度。两者一除就是"X每变动一单位Y平均变动多少"。

(2) 无偏性证明

Step 1：代入真实关系。做变换 Yi − Ȳ = β₂(Xi − X̄) + (μi − μ̄)，代入β̂₂的分子：

Σ(Xi−X̄)(Yi−Ȳ) = β₂·Σ(Xi−X̄)² + Σ(Xi−X̄)·μi
（其中 Σ(Xi−X̄)·μ̄ = μ̄·Σ(Xi−X̄) = 0，离差和为零）

Step 2：除以分母 Σ(Xi−X̄)²：
β̂₂ = β₂ + Σ(Xi−X̄)·μi / Σ(Xi−X̄)²
（"真值 + 误差项"的结构）

Step 3：两边取期望，由假定 E(μi|X) = 0：
E(β̂₂|X) = β₂ + 0 = β₂

故β̂₂是β₂的无偏估计量。 □

知识点总结

⚠ 关键依赖 E(μi|X)=0。若 Cov(X,μ)≠0（内生性），Step 3崩塌，β̂₂有偏且不一致。

矩阵形式等价：β̂=(X'X)⁻¹X'Y。考试给一元方程用标量形式，给矩阵方程用矩阵形式。

Q5 多重共线性对计量结果的影响有哪些？ 简答 ★★★
展开答案

答案

多重共线性是指解释变量之间存在高度线性相关。OLS估计量仍然无偏且一致，但方差膨胀（VIF增大），导致t统计量偏小，单个变量的显著性检验容易失灵。但模型整体的F检验和R²不受影响，因为变量合在一起的解释力没减少，只是无法区分各自的贡献。

判断方法：① |rᵢⱼ|≈1；②VIF>10；③F大R²大但t普遍小；④CI>20。

补救：剔除不重要变量、逐步回归（AIC/BIC选模型）、岭回归。

知识点总结

⚠ 完全多重共线性（如X₁+2X₂=0）→ (X'X)不可逆 → 无法求出OLS估计量。

⚠ 虚拟变量陷阱：n个类别只能引入n−1个虚拟变量，引入n个→完全共线性。

Q8 什么是内生性？如何检验和补救？工具变量的条件是什么？ 简答 ★★★★
展开答案

答案

内生性是指解释变量X与随机误差项U相关，即Cov(X,U)≠0。常见来源：遗漏变量、双向因果、测量误差。

内生性是所有假设违反中最严重的——直接破坏无偏性证明中E[μi|X]=0这一步，导致β̂有偏且不一致（样本量再大也没用）。

检验：Hausman检验，H₀为"不存在内生性"，拒绝则改用工具变量法。

补救：工具变量法(IV)/2SLS。Z必须满足：①相关性 Cov(Z,X)≠0；②外生性 Cov(Z,U)=0；③排斥性（Z不直接出现在原方程中）。三条缺一不可。

2SLS步骤：第一阶段X对Z回归得X̂；第二阶段用X̂替代X对Y做OLS。

Q10 什么是受约束F检验？怎么构造？ 推导/计算 ★★★★
展开答案

答案

受约束F检验用于检验一组约束条件是否成立。思路：加约束后拟合变差（RSS变大），变差幅度相对误差水平足够大就说明约束是错的。

F = [(RSS_R − RSS_UR) / q] / [RSS_UR / (n−k−1)]

当两个模型Y相同时（TSS相同），可用R²形式：
F = [(R²_UR − R²_R) / q] / [(1−R²_UR) / (n−k−1)]

整体F检验是特例（约束=所有斜率=0）。Chow检验也是特例（约束=两段系数相同）。

⚠ R²形式使用条件：两个模型的Y完全相同。Y被变换了就不能用。

Q11 怀疑某个时间点前后模型参数变了，怎么检验？ 简答
展开答案

答案

用Chow检验。思路：如果两段参数相同，合在一起回归和分开回归的拟合效果应该差不多。把数据按怀疑的时间点分成前后两段，分别OLS得RSS₁和RSS₂，全样本合并回归得RSS_pooled。如果合并后RSS比分开时大很多，说明参数确实不同。

F = [(RSS_pooled − RSS₁ − RSS₂) / (k+1)] / [(RSS₁ + RSS₂) / (n − 2(k+1))]

H₀：两段参数相同。拒绝H₀ → 存在结构突变。

Chow检验本质是受约束F检验的特例。

Q12 名词解释：虚拟变量陷阱 名词解释
展开答案

若定性变量有n个类别，建模时应只引入n−1个虚拟变量。如果引入了n个，这n个虚拟变量之和恒等于1（与截距项完全共线），导致(X'X)不可逆，无法求出OLS估计量。这种因虚拟变量设置不当导致完全多重共线性的问题称为虚拟变量陷阱。

答案

EG两步法检验两个非平稳序列是否存在协整（长期均衡关系）。

第一步：OLS回归 Y=α+βX+û，得到残差û。第二步：对û做ADF检验。拒绝H₀（û平稳）→存在协整；不拒绝→无协整，回归是伪回归。

知识点总结

协整：两个非平稳序列的某个线性组合是平稳的。伪回归：无协整就做回归，R²高但无意义（症状：R²>DW）。一阶用EG检验，多阶用Johansen检验。

Q9 误差修正模型的主要作用是什么？ 简答 ★★★
展开答案

ECM解决两个变量的短期失衡。存在协整（长期均衡）时，短期可能偏离，ECM通过误差修正项让偏离在下一期得到修正。

ΔYₜ = α·ûₜ₋₁ + 短期动态项 + vₜ（α<0是纠偏机制，|α|越大修正越快）

建立步骤：第一步建长期关系模型（协整方程），第二步建短期动态关系（误差修正方程）。

Q10 名词解释：随机游走 / 伪回归 / 内生变量 名词解释
展开答案

随机游走：Yₜ=Yₜ₋₁+εₜ（εₜ白噪声）。当前值=上期值+随机冲击。完全随机、不可预测、无长期均值。Var(Yₜ)=tσ²随时间增长→非平稳。AR(1)中φ=1的特例。

伪回归：非平稳序列不存在协整关系就做回归，R²高但无统计意义。

内生变量：与误差项相关的解释变量，Cov(X,U)≠0。在联立方程中由系统内部联合决定。

Q11 联立方程模型：指出内生/外生变量，写出简化式模型 推导
(1) Yₜ = β₀ + β₁Mₜ + γ₁Cₜ + γ₂Iₜ + μ₁ₜ

(1) 变量分类

内生变量（互相决定）：Yₜ、Mₜ。外生变量：Cₜ、Iₜ、Pₜ。先决变量={Cₜ, Iₜ, Pₜ}。

(2) 简化式推导

导出Yₜ的简化式：将方程(2)代入(1)消去Mₜ：

Yₜ = β₀ + β₁(α₀ + α₁Yₜ + γ₃Pₜ + μ₂ₜ) + γ₁Cₜ + γ₂Iₜ + μ₁ₜ

移项：Yₜ(1−β₁α₁) = (β₀+β₁α₀) + γ₁Cₜ + γ₂Iₜ + β₁γ₃Pₜ + (μ₁ₜ+β₁μ₂ₜ)

两边除以(1−β₁α₁)：
Yₜ = (β₀+β₁α₀)/(1−β₁α₁) + γ₁/(1−β₁α₁)·Cₜ + γ₂/(1−β₁α₁)·Iₜ + β₁γ₃/(1−β₁α₁)·Pₜ + v₁ₜ

ECMError Correction ModelΔYₜ = α·ûₜ₋₁ + 短期项 + vₜ有协整时的短期调整机制，α<0为纠偏
ECM 误差修正模型 协整配套
定义在协整关系确立后，用来描述变量偏离长期均衡后如何在短期内调整回来的模型。
ΔYₜ = α · ûₜ₋₁ + Σγᵢ·ΔYₜ₋ᵢ + Σδⱼ·ΔXₜ₋ⱼ + vₜ
关键参数 αα < 0（纠偏系数）：当上期偏离均衡（ûₜ₋₁ > 0），本期Y会向下调整，速度由|α|决定。
若 α ≥ 0，偏离不会收敛，ECM无意义。
ûₜ₋₁ 的来源EG第一步回归的残差：ûₜ₋₁ = Yₜ₋₁ − β̂₀ − β̂₁Xₜ₋₁，代表上期偏离长期均衡的程度。
前提条件Y和X必须协整（EG检验或Johansen检验通过）。无协整则不能建ECM。
格兰杰表示定理协整 ↔ ECM表示：两个协整的I(1)序列，一定可以表示为ECM；反之亦然。
σ²ₜ条件方差Var(εₜ | 过去信息)方差不再是常数，随时间变化——ARCH/GARCH建模对象
σ²ₜ 条件方差（时变方差）方差模型
定义方差不再是常数，而是依赖于过去信息的时变量：σ²ₜ = Var(εₜ | 过去信息集 Ωₜ₋₁)。
为什么需要OLS假定同方差（Var(εᵢ)=常数），但金融序列常出现波动聚集：大波动后跟大波动，小波动后跟小波动。这违反了同方差假定。
波动聚集直觉股市崩盘后，连续多天大幅震荡（高波动）；平静期连续多天微幅波动（低波动）。波动是可预测的。
建模思路先建均值方程（AR/MA/ARMA），提取残差 εₜ；对残差平方 ε²ₜ 检验是否有自相关（ARCH-LM检验）；有ARCH效应 → 建 ARCH/GARCH。
ARCH(q)Autoregressive Conditional Heteroscedasticityσ²ₜ = α₀ + α₁ε²ₜ₋₁ + … + αqε²ₜ₋q用历史残差平方预测当前方差
ARCH(q) 自回归条件异方差模型 方差模型
定义上面的 AR/MA/ARMA 都在建模均值（Y的期望值随时间怎么变）。ARCH 建模的是方差（Y的波动大小随时间怎么变）。
σ²ₜ = α₀ + α₁ε²ₜ₋₁ + α₂ε²ₜ₋₂ + … + αqε²ₜ₋q
参数识别方差方程里只有 ε² 项（历史残差平方），数有几个就是 q。例：4个 ε²ₜ₋ᵢ → ARCH(4)。
核心直觉昨天波动大（ε²大）→ 今天也倾向于波动大。金融市场的"波动聚集"现象。
参数约束α₀ > 0，αᵢ ≥ 0（方差不能为负），Σαᵢ < 1（保证弱平稳）。
检验方法ARCH-LM检验：对残差平方做辅助回归，F统计量显著 → 存在ARCH效应。
使用场景金融收益率序列（股票、汇率、利率）普遍存在波动聚集。
GARCH(p,q)Generalized ARCHσ²ₜ = ω + αε²ₜ₋₁ + βσ²ₜ₋₁ARCH + 方差自身滞后项，ARMA用在方差上
GARCH(p,q) 广义ARCH模型 方差模型
定义在 ARCH 基础上，方差方程里再加上 σ² 自身的滞后项。相当于把 ARMA 的思想用在方差上：ARCH = AR用在方差上，GARCH = ARMA用在方差上。
σ²ₜ = ω + α₁ε²ₜ₋₁ + … + αqε²ₜ₋q + β₁σ²ₜ₋₁ + … + βₚσ²ₜ₋ₚ
参数识别方差方程里有 ε² 项（q个）+ σ² 项（p个）。注意：p是σ²的个数，q是ε²的个数。例：6个σ² + 4个ε² → GARCH(6,4)。
最常用GARCH(1,1)：σ²ₜ = ω + αε²ₜ₋₁ + βσ²ₜ₋₁。实证中绝大多数情况够用。
弱平稳条件α + β < 1；且 ω > 0，α ≥ 0，β ≥ 0。（考试高频！）
使用场景金融波动率建模的标准工具。ARCH-LM检验拒绝H₀后，通常直接建GARCH(1,1)。
ARCH vs GARCH ARCH(q)：只看历史冲击 ε²
GARCH(p,q)：ε² + σ² 自身滞后
关系：ARCH(q) = GARCH(0,q)
均值模型 vs 方差模型 AR/MA/ARMA → 建模 E(Yₜ)
ARCH/GARCH → 建模 Var(εₜ)
实践中两者经常组合使用

A. 概率论基础 方差 · 协方差 · 相关系数 · 分布 · 分位点
方差 Var(X) 概率论
Var(X) = E[(X − μ)²]
衡量随机变量 X 围绕均值波动的程度。方差越大，取值越分散；方差越小，取值越集中。
直觉：月薪每次都是8000元 → 方差=0；月薪在5000到11000之间随机变化 → 方差大。
在计量中：AR(1)的 Var(Yₜ) = σ²_ε/(1−φ²)，φ越接近1，方差越大，序列越不稳定。当φ=1时分母为零，方差趋于无穷，这正是随机游走不平稳的原因。
协方差 Cov(X, Y) 概率论
Cov(X, Y) = E[(X − μₓ)(Y − μᵧ)]
特殊情况：Cov(X, X) = Var(X)
衡量两个变量 X 和 Y 共同变动的方向：
Cov > 0：X 大时 Y 也大（同向）  |  Cov < 0：X 大时 Y 反而小（反向）  |  Cov = 0：无线性关系
问题：协方差只能说明方向，无法说明"多强"，且受量纲影响（人民币 vs 美元没有可比性）→ 需要标准化成相关系数。
在时间序列中：自协方差 γₖ = Cov(Yₜ, Yₜ₋ₖ)→ 协方差，是序列自己和自己滞后k期的协方差，ACF公式的分子就是它。
相关系数 ρ(X, Y) 概率论
ρₓᵧ = Cov(X, Y) / (σₓ · σᵧ)　　取值范围：[−1, 1]
协方差除以各自的标准差，消除量纲差异，得到"标准化的协方差"。ρ=1：完全正相关；ρ=−1：完全负相关；ρ=0：无线性相关。
为什么要除以标准差？猪肉价格涨10元（人民币）和牛肉价格涨5美元，直接比协方差没意义，单位不同。除以标准差后变成纯数字，才能比较。
ACF的联系：ACF公式 ρₖ = γₖ / γ₀ 就是对自协方差做了标准化，分母用的是γ₀（即Yₜ的方差）而不是各自标准差的乘积——这正是ACF和PACF的核心区别。
标准差 σ 概率论
σ = √Var(X)
方差的平方根，把方差恢复到原数据的单位，让波动程度可以和数据值直接比较。
例：股价（元）的方差单位是"元²"，不直观。开根号变成标准差，单位是"元"，可以直接说"在均值±50元范围内波动"。

正态分布 N(μ, σ²)
钟形曲线，关于μ对称。

大样本下统计量趋近正态。

类似正态，尾部更厚（小样本）。

右偏，只取正值。
检验多个系数联合显著性：
联合显著性、Granger因果。

分位点（临界值）的含义 概率论
F分布上α分位点：P(X > Fα) = α　（右尾面积恰好等于α）
t分布双侧分位点：P(|X| > t_{α/2}) = α　（两尾各α/2）
直觉：在分布曲线上找一个点，使右边的面积恰好等于α。这个点就是临界值。α就是"右边那一小块的面积"。
假设检验的应用：若统计量落在临界值右边（进入拒绝域），意味着"在H₀成立时，出现这么极端值的概率不足α"→ 拒绝H₀。
t检验简化：t分布天然双侧，对统计量取|t|后可转为单侧，和F检验逻辑完全一样：|t| > 临界值 → 拒绝H₀。
帽子符号 β̂ 的含义 统计
β → 真实参数（总体，永远未知）　　β̂ → β 的估计值（由样本数据算出）
在统计学中，真实值和估计值的区分非常重要。带帽子 = 用样本估计出来的，软件输出的都是估计值。
类比：想知道全中国人均身高μ（真实值，测不完），只能抽样算出μ̂（估计值）。μ̂ ≠ μ，但估计方法好的话，μ̂ 应该接近 μ。
B. 线性代数基础 矩阵 · 转置 · 逆矩阵 · 满秩
矩阵与向量 线性代数
数据矩阵 X（n×k）：n行 = n个观测，k列 = k个解释变量
回归方程矩阵形式：Y = Xβ + u
矩阵就是多个向量的集合，把所有数据整理成统一的形式，方便推导公式。用矩阵写出的回归方程比写n个单独方程简洁得多。
矩阵转置 X' 线性代数
X 是 n×k → X'（转置）是 k×n　（行变列，列变行）
结果：X'X 是 k×k 方阵；X'Y 是 k×1 向量
维度检查：(X'X)⁻¹是(k×k)，X'Y是(k×1)，所以 β̂ = (X'X)⁻¹X'Y 维度是 k×1，正好是k个参数的向量。
逆矩阵 A⁻¹ 与 OLS 公式来源 线性代数
A · A⁻¹ = I（单位矩阵）　← 矩阵版的"除法"
逆矩阵是矩阵的"倒数"，两者相乘等于单位矩阵（对角线全1，其余为0）。OLS公式就是对回归方程左乘逆矩阵推导出来的：
Y = Xβ + u
→ X'Y = X'Xβ　（两边左乘X'）
→ (X'X)⁻¹X'Y = β　（两边左乘(X'X)⁻¹）
→ β̂ = (X'X)⁻¹X'Y
⚠ 逆矩阵存在的条件：矩阵必须满秩。如果X'X不可逆（不满秩）→ OLS公式无法计算 → 这就是完全多重共线性的本质。
满秩条件 线性代数
满秩：矩阵中所有列向量线性无关
满秩 → X'X 可逆 → β̂ = (X'X)⁻¹X'Y 能算出来
如果某个X列可以被其他X列的线性组合表示出来，那这一列就是"多余的"，不提供新信息，矩阵不满秩。
不满秩的情形（完全多重共线性）：比如 X₃ = 2X₁ + X₂，X₃可以完全由X₁和X₂推出来 → X'X不可逆 → OLS崩溃，算不出β̂。
C. 统计推断基础 假设检验 · P值 · 估计量性质
假设检验基本框架 统计
H₀（原假设）：通常是"没有效果"，如 β=0
H₁（备择假设）：通常是"有效果"，如 β≠0

核心口诀：数大率小拒绝H₀ | 数小率大不拒绝H₀
先假设H₀成立，看数据是否提供了足够强的证据来推翻它。"足够强"的标准：统计量超过临界值（等价于P值小于α）。
⚠ 只能说"不拒绝H₀"，不能说"接受H₀"。未能推翻 ≠ 证明正确，就像未定罪 ≠ 无罪。
P值是什么 统计
P值 = 假设H₀为真时，得到"比当前数据更极端的结果"的概率
P值越小 → 当前数据在H₀成立下出现的概率越低 → H₀越可能是错的 → 越倾向于拒绝H₀。
P < 0.05 → 在5%水平下拒绝H₀（最常用标准）
P > 0.1 → 证据不足，不拒绝H₀
⚠ P值不是"H₀正确的概率"！是"假设H₀正确时，出现这么极端数据的概率"。两者含义完全不同。
OLS估计量的三大性质 统计
无偏性　E(β̂) = β
平均来说等于真实值，没有系统偏差
有效性　Var(β̂) 最小
在所有无偏估计量中方差最小（最精确）
一致性　n→∞ 时 β̂ → β
样本越大，越接近真实值
BLUE　无偏 + 最小方差
高斯假定满足时OLS同时具备
类比射箭：无偏=落点平均在靶心（无偏差）；有效=落点集中（散布小）；一致=箭越多越接近靶心。BLUE=既不偏又集中，最优。
核心口诀：数大率小拒绝H₀ 统计
数大率小，拒绝H₀  |  数小率大，不拒绝H₀
"数" = 统计量 vs 临界值（谁大）  "率" = P值 vs α（谁小）
以F分布为例：在曲线上找临界点，右侧面积 = α（拒绝域）。统计量落在临界值右边 → 进入拒绝域 → 拒绝H₀。
统计量法
统计量 > 临界值 → 拒绝H₀
统计量 ≤ 临界值 → 不拒绝
P值法
P值 < α → 拒绝H₀
P值 ≥ α → 不拒绝
⚠ 两种判断方式完全等价，选一种用即可。
α 的三种理解角度 统计
角度1：小概率事件的门槛
H₀代表"本来不太可能发生"的事。α是你允许它发生的最低概率。P值 < α → 连门槛都没达到 → 拒绝H₀。
角度2：弃真概率的上限（学术定义）
α = "H₀为真时，我错误拒绝它的最大允许概率"。P值 < α → 弃真概率极小 → 拒绝H₀是安全的。
角度3：约会比喻（最好记）
H₀ = 当前约会对象；α = 你心中的最低标准；对方表现（P值）< 你的标准（α）→ 拒绝H₀，考虑换人。
⚠ 结论只能说"不拒绝H₀"，不能说"接受H₀"——H₁从未被直接检验过。
单侧检验 vs 双侧检验 统计
类型H₀ / H₁写法拒绝域临界值
双侧H₀: β=0   H₁: β≠0两侧各α/2t(α/2)
单侧（左）H₀: β≥0   H₁: β<0左侧α−t(α)
单侧（右）H₀: β≤0   H₁: β>0右侧αt(α)
t检验天然双侧；实操简化：对|t|取绝对值转单侧，统一用α分位点。F检验天然单侧（F值≥0）。
⚠ 不同教材临界值取法不同（α/2双侧 vs α单侧），本质一样，可互相转化。
假设检验四步流程（万能模板）统计
第一步：列假设 — 写出H₀（通常是"没有效果"，如β=0）和H₁（研究者想证明的）。
第二步：构造统计量
t = β̂ / SE(β̂) → t分布（检验单个系数）
F = (RSS_R − RSS_U)/q ÷ RSS_U/(n−k−1) → F分布（检验多个约束）
χ² = nR² → 卡方分布（LM检验）
第三步：查临界值 — 根据α和自由度查表。t检验取|t|转单侧查t(α)；F检验查F(α)。
第四步：下结论 — 统计量 > 临界值 → 拒绝H₀；否则不拒绝。给出经济含义解释。

D1. 模型设定与假定

计量分析基本思路 + 静态模型条件 计量
Y = f(X) + ε　可观测部分：X；不可观测：扰动项ε

ε的条件：白噪声（零均值+同方差+无自相关+正态）
违反 → 失去最小方差无偏性
误差 vs 残差——符号精确区分 计量
误差 U（Error）
符号：ε 或 u
真实值 − 真实回归线
总体角度，永远不可观测
残差 e（Residual）
符号：eᵢ 或 ûᵢ
真实值 − 估计回归线
样本角度，OLS算完可得
误差u = "上帝知道的真实偏离"；残差ê = "我们能算出的偏离估计"。残差是误差的估计值。
残差的四条假定（正态白噪声条件）计量
#假定公式违反后果
①零均值E(εᵢ) = 0估计有系统偏差
②同方差Var(εᵢ) = σ²异方差 → OLS不再BLUE
③无自相关Cov(εᵢ, εⱼ) = 0自相关 → 标准误不准

TSS 分解与拟合优度 R² 计量
三段差：离差 = 回归差 + 残差
Yᵢ−Ȳ = (Ŷᵢ−Ȳ) + (Yᵢ−Ŷᵢ)

平方和分解：TSS = ESS + RSS
TSS = Σ(Yᵢ−Ȳ)²　总变动
ESS = Σ(Ŷᵢ−Ȳ)²　模型解释了的部分（回归平方和）
RSS = Σ(Yᵢ−Ŷᵢ)²　模型没解释的部分（残差平方和）

R² = ESS/TSS = 1 − RSS/TSS　∈ [0,1]
R²=1：RSS=0，所有点在回归线上 → 完美拟合

ESS直觉：最烂的模型Ŷ=Ȳ → ESS=0；好模型Ŷ随X变化 → ESS大，捕捉到规律
⚠ R²随解释变量增加单调不减（加无用变量R²也升）→ 多变量比较时用调整R²（Adj-R²）
⚠ AIC = ln(σ̂²) + 2k/n；BIC = ln(σ̂²) + k·ln(n)/n，越小越好，BIC惩罚更重

含义：Y有惯性记忆

D2. 平稳性与白噪声

白噪声的三个条件与分类 计量
① E(εₜ) = μ（均值常数）
② Var(εₜ) = σ²（同方差）
③ Cov(εₜ, εₛ) = 0，t≠s（无自相关）
满足①②③ → 白噪声；+正态 → 正态白噪声；+零均值 → 零均值正态白噪声；+方差=1 → 标准正态白噪声。
建模目标：把残差变成白噪声，说明序列里的"可预测部分"已被模型完全抓走。
平稳性三层次：严格平稳 > 白噪声 > 宽平稳 计量
层次核心要求是否允许自相关
严格平稳任意时刻联合分布完全相同取决于分布
白噪声均值常数+同方差+无自相关❌ 不允许
宽平稳均值不变+协方差只与间隔有关✓ 允许
关键：宽平稳允许自相关 → 时序建模利用这个自相关；白噪声无自相关 → 是残差检验的目标。
为什么用对数差分 ≈ 收益率？ 计量
ln(Pₜ) − ln(Pₜ₋₁) = ln(Pₜ/Pₜ₋₁) = ln(1 + rₜ) ≈ rₜ（当rₜ很小时）
绝对差分无法跨股票比较 → 收益率rₜ消除量纲 → 对数差分≈rₜ且计算更简便（连加代替连乘）。同时通常使序列从I(1)变为平稳的I(0)。
⚠ 近似仅在小幅波动时精确；大幅波动（如崩盘）时误差增大。

D3. 单位根与AR结构

AR(1)中φ的作用：为什么φ=1导致不平稳？ 计量
AR(1)展开：Yₜ = φᵗ·Y₀ + Σₖ φᵏ·εₜ₋ₖ　（每期冲击以φᵏ衰减到今天）
φ的值φᵏ的行为结论
|φ| < 1随k增大→0（衰减）冲击消散 → 平稳 ✓
φ = 1永远=1（不衰减）冲击累积，Var→∞ → 不平稳 ✗
|φ| > 1越来越大（放大）冲击放大 → 更不平稳 ✗
滞后算子 L：高阶AR的统一分析工具 计量

每个变量的当期值 = 自己和对方历史值的线性组合。配套工具：IRF（脉冲响应）、FEVD（方差分解）。
伪回归 vs 协整 计量
伪回归：两个不平稳序列直接OLS → R²虚高、t显著，但无真实关系。根因：拟合的是"共同趋势"。
协整：线性组合 Z = aY + bX 是平稳的 → 虽然各自不平稳，但存在长期均衡关系。
⚠ 协整要求各序列是相同阶数的单整（通常都是I(1)）。
EG 两步法：检验两序列是否协整 计量
第一步：OLS回归 Y = β̂₀ + β̂₁X，得残差û（提取偏离长期均衡的部分）
第二步：对û做ADF检验。H₀：û不平稳（无协整）；H₁：û平稳（有协整）。
⚠ 此处ADF临界值与普通ADF不同！残差是估计量而非真实值，需用特殊临界值。
格兰杰因果：从系数判断四种情况 计量
Yₜ = Σαᵢ·Xₜ₋ᵢ + Σβᵢ·Yₜ₋ᵢ + μ₁ₜ　（α是X→Y的交叉项）
Xₜ = Σλᵢ·Yₜ₋ᵢ + Σδᵢ·Xₜ₋ᵢ + μ₂ₜ　（λ是Y→X的交叉项）
αλ结论
≠0=0单向：X → Y
=0≠0单向：Y → X
≠0≠0双向：X ⇌ Y
=0=0独立，互不预测
⚠ 格兰杰因果 ≠ 真实因果，只是"预测能力"。H₀：交叉项全为零，F检验。
